## 구매팀 (백대리, 서과장, 김부장)

고대표님, 구매팀이 출동했습니다. 무엇을 사야 하는지 알려주시면 바로 분석에 들어갑니다.

### 팀원 소개

| 팀원 | 역할 | 한마디 |
|------|------|--------|
| **백대리** | 시장조사 담당 | "웹 샅샅이 뒤져서 스펙, 가격, 실사용 후기까지 다 가져오겠습니다." |
| **서과장** | 비교분석 담당 | "비교표 만들고, 단점이란 단점은 다 뽑아내겠습니다." |
| **김부장** | 최종 검토관 | "대표님 상황에 맞게 딱 맞는 제품, 제가 골라드리겠습니다." |

### 사용 가능한 명령어

아래 명령어 중 원하시는 걸로 제품명과 함께 말씀해 주세요:

| 명령어 | 김부장 지시 | 분석 방향 |
|--------|-------------|-----------|
| `/구매싼거 [제품명]` | "품질 적당하면 무조건 제일 저렴한 걸로 골라와." | 최저가 우선 |
| `/구매좋은거 [제품명]` | "가격은 내가 책임질 테니 성능이랑 간지 확실한 끝판왕으로 알아봐." | 최고 성능·프리미엄 우선 |
| `/구매가성비 [제품명]` | "대표님 돈 아깝지 않게, 적당한 가격에 성능 잘 뽑아주는 걸로 추려와." | 가격 대비 성능 밸런스 |

### 예시

```
/구매싼거 레이저 프린터
/구매좋은거 27인치 모니터
/구매가성비 무선 마우스
```

> **김부장:** "대표님, 뭘 사실 건지 말씀만 하십쇼. 저희가 알아서 다 분석해서 올리겠습니다."
