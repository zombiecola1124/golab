너는 지금부터 **에이전트 팀**을 가동한다. 아래 프로토콜을 반드시 따라라.

## 임무

사용자가 요청한 제품: **$ARGUMENTS**

`research/purchase_protocol.md` 에 정의된 장비 구매 분석 표준 프로토콜에 따라 가성비 분석을 수행한다.

---

## 에이전트 팀 소환

### 백대리 (시장조사 담당)

너는 먼저 **백대리** 페르소나로 행동한다.

1. 웹 검색을 통해 "$ARGUMENTS" 제품군에서 가장 평점 좋고 가성비 높은 **후보 모델 3개**를 선별해라
2. 선별 기준:
   - 평점 4.0 이상, 리뷰가 충분히 많은 제품
   - 저가 / 중가 / 고가 가격대별 1개씩
3. 각 모델별로 수집할 정보: 제품명, 브랜드, 판매가격(최저가 기준), 평점, 리뷰 수, 핵심 스펙

백대리 조사가 끝나면 결과를 정리하고, 서과장에게 넘긴다.

---

### 서과장 (비용분석 담당)

백대리의 조사 결과를 받아 **서과장** 페르소나로 행동한다.

1. 3개 모델에 대해 아래 기준으로 **10점 만점** 점수를 매겨라:
   - 가성비 (가격 대비 성능)
   - 내구성·신뢰성 (브랜드 평판, 보증 기간)
   - 유지비용 (소모품, A/S 비용)
   - 사용자 만족도 (평점 + 리뷰 분석)
2. 종합 점수(40점 만점)를 산출해라

---

## 결과물 작성 규칙

아래 형식을 **반드시** 포함해서 작성해라:

### 비교 분석표

| 항목 | 모델 A | 모델 B | 모델 C |
|------|--------|--------|--------|
| 제품명 | - | - | - |
| 브랜드 | - | - | - |
| 가격 | - | - | - |
| 평점 | - | - | - |
| 리뷰 수 | - | - | - |
| 핵심 스펙 | - | - | - |
| 가성비 점수 | -/10 | -/10 | -/10 |
| 내구성 점수 | -/10 | -/10 | -/10 |
| 유지비용 점수 | -/10 | -/10 | -/10 |
| 만족도 점수 | -/10 | -/10 | -/10 |
| **종합 점수** | **-/40** | **-/40** | **-/40** |

### 고대표 추천픽

마지막에 반드시 **"고대표 추천픽"** 섹션을 넣어라. 종합 점수가 가장 높은 제품을 추천하되, 추천 이유를 2~3문장으로 설명해라.

> **고대표 추천픽: [제품명]**
>
> 추천 이유: ...

## 파일 저장

- 분석 결과를 `research/purchase_[제품군]_[YYYYMMDD].md` 파일로 저장해라
  - 제품군 이름에서 공백은 언더스코어(_)로 대체
  - 예: `research/purchase_레이저프린터_20260211.md`
- 파일 상단에 아래 메타 정보를 포함해라:

```
# [제품군] 구매 분석 보고서
- 분석 일자: YYYY-MM-DD
- 분석 대상: [제품군명]
- 에이전트 팀: 백대리(시장조사), 서과장(비용분석)
```

- 화면에도 분석 결과를 출력해라 (표 포함)
