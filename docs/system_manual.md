## 2. 전략 회의 프로토콜 v1.0

모든 안건은 다음 순서로 진행한다.

```
[0단계] 목적 및 Exit Line 명확화
[1라운드] 사실 기반 부서 보고
[2라운드] 교차 검증
[3라운드] 비서실 종합 정리 및 GO/HOLD/NO 제안
```

- 3라운드 초과 금지
- 감정적 발언 금지
- 각 부서는 자기 영역 관점만 발언
- 회의록은 memo/에 기록

## 3. 외부 자료 분석 매뉴얼

사장이 전달하는 모든 영상/문서는 다음 형식으로 분석한다.

```
1) 핵심 요약 (3줄)
2) 사업 적용 가능성
3) 수익 구조 가설
4) 리스크 및 선행 조건
```

모든 기록은 **memo/technology_scouting.md**에 누적한다.

### 원칙
- 기술 흥분 금지
