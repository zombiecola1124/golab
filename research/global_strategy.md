## 1. 기획 배경

**대표님 지시 요약:**
- 스트라이프 애틀라스(Stripe Atlas)를 활용한 미국 법인 설립 전략 검토
- 글로벌 결제 시스템(Stripe) 구축 가능성 평가
- 일본 연마기 사업 및 SaaS 프로젝트의 글로벌 확장 전략 수립
- 참고: 코드팩토리 미국 법인 설립기

**스트라이프 애틀라스란?**
- Stripe이 제공하는 미국 법인 설립 원스톱 서비스
- 140개국+ 창업자가 Delaware주에 C-Corp 또는 LLC를 $500에 설립 가능
- 법인 설립 → EIN 발급 → Mercury 은행 계좌 → Stripe 결제 활성화까지 올인원
- 2025년 기준: 169개국에서 23,000개 기업이 Atlas로 설립, Delaware C-Corp의 5개 중 1개가 Atlas 출신
- 100% 온라인 진행, 미국 방문 불필요, SSN 불필요

## 2. 트럼프 실익 분석 — 재무 검토

### 2-1. 비용 구조 (숫자로 말한다)

#### 초기 비용

| 항목 | 금액 | 비고 |
|------|------|------|
| Stripe Atlas 법인 설립 | **$500** (약 70만원) | 레퍼럴 할인 시 $300 |
| Delaware 주 filing fee | $90 | Atlas $500에 포함 |
| 1년차 등록 대리인 | 무료 | Atlas에 포함 |
| 미국 전화번호 (eSIM) | $1~10/월 | Mint Mobile 등 |
| 미국 가상 주소 | $10~30/월 | Stable 등 |
| **초기 총 비용** | **약 $530~$570** | **약 75~80만원** |

#### 연간 유지비 (LLC 기준)

| 항목 | 금액 | 기한 |
|------|------|------|
| 등록 대리인 | $100/년 | 매년 |
| Delaware LLC 연간 세금 | **$300/년** | 매년 6월 1일 |
| Delaware 사업자 면허 | $75/년 | 매년 |
| 미국 세금 신고 (Form 5472 + 1120) | $300~600/년 | 4월 15일 (회계사) |
| 전화번호 유지 | $12~120/년 | 매월 |
| 가상 주소 | $120~360/년 | 매월 |

| 한-미 양국 세무 대리 이용 시 | +$1,500~$3,000/년 | 약 400~800만원 추가 |

> **트럼프 코멘트:** "매출이 $0이어도 연 127만원은 나간다. 숫자가 안 맞는 비즈니스에 매년 돈을 태우는 건 바보짓이야. 연 매출 $50,000(약 7,000만원) 이상이 확보되기 전에는 법인 설립하면 안 된다."

### 2-2. 한국 법인 vs 미국 LLC 세무 비교

| 항목 | 한국 법인(개인사업자) | 미국 LLC (Delaware) |
|------|----------------------|---------------------|
| **법인세/소득세** | 6.6~49.5% (종합소득세+지방세) | LLC 자체 과세 없음 (Pass-through) |
| **부가세** | 10% VAT | 미국 Sales Tax (주마다 다름, 디지털은 대부분 면제) |
| **결제 수수료** | 3.5~4.5% (토스, NicePay 등) | 2.9% + $0.30 (Stripe 미국 내) |
| **국제 결제 수수료** | 4.5~6% (한국 PG 국제결제) | 3.9% + $0.30 (Stripe 국제) |
| **글로벌 확장** | 한국 PG로는 해외 결제 수취 매우 제한적 | 195개국에서 결제 수취, 135개 통화 지원 |
| **투자 유치** | 한국 VC만 가능 | 글로벌 VC 접근 가능 (C-Corp 전환 시) |

### 2-3. 달러 자산 보유 장단점

**장점:**
- 원화 약세 시 자산 가치 보전 (2025~2026 원/달러 1,400~1,500원대 "뉴노멀")
- 글로벌 서비스 결제(AWS, 도메인, SaaS 도구) 시 환전 비용 절감
- 통화 분산 효과

**단점:**
- 양방향 환율 리스크 (원화 강세 시 손실)
- 송금 수수료 0.5~1.5%/건
- 해외금융계좌 신고 의무 (5억원 초과 시)
- 외환거래법 신고 의무

> **트럼프 판정:** "달러 자산 보유 자체는 나쁘지 않다. 하지만 이건 돈을 벌고 나서 할 고민이야. 매출도 없는데 달러 자산 운운하는 건 집 짓기 전에 인테리어 걱정하는 거다."

## 3. 박이사 시장 전략 — 글로벌 결제 우위

### 3-1. 일본 연마기 B2B 사업

**Stripe가 주는 우위:**

| 기능 | 효과 |
|------|------|
| 다통화 인보이스 | 일본 바이어에게 JPY 청구 → 자동으로 USD 수취. 환전 자동화 |
| 자동 인보이싱 | Net 30, Net 60 등 B2B 결제 조건 설정 가능. 수동 청구서 작성 불필요 |
| 반복 결제 | 소모품(연마석, 부품) 정기 납품 시 자동 청구 가능 |
| 미국 법인 신뢰도 | "US-based company" 표기로 국제 B2B 거래 시 신뢰도 상승 |
| USDC 결제 | 2025년 신규 — 스테이블코인으로 2~5일 걸리던 정산이 수 분 내 완료 |

**단, 현실 체크:**
- 현재 일본 연마기 사업은 주로 **오프라인 B2B 거래**로, Stripe 온라인 결제가 당장 필수는 아님
- 기존 거래처는 이미 은행 송금(T/T) 기반 — 결제 시스템 전환 동기가 낮을 수 있음
- Stripe는 **온라인 결제**에 최적화, 오프라인 B2B 대금 결제에는 과잉 스펙

### 3-2. SaaS 프로젝트 글로벌 확장

**여기가 진짜 핵심이다.** Stripe가 압도적 우위를 보이는 영역:

| 기능 | Stripe | 한국 PG (토스, NicePay 등) |
|------|--------|---------------------------|
| 구독 과금 (월/연간) | 7가지+ 과금 모델 지원 | 제한적 |
| 글로벌 결제 수취 | 195개국, 135개 통화 | 사실상 국내 전용 |
| 세금 자동화 | 100개국+ VAT/GST 자동 계산 (Stripe Tax) | 없음 |
| 결제 승인률 | 높음 (현지 acquirer 연결) | 해외 카드 → 높은 거부율 |
| Apple Pay / Google Pay | 기본 지원 | 제한적 |
| 개발자 API | REST API, 풍부한 SDK | 복잡한 연동, PG사마다 다름 |

**전환율 데이터 (이박사 팩트 체크 — 신뢰도: 중):**
- Stripe Checkout 사용 시 경쟁사 대비 **+8% 전환율** (Stripe 자체 A/B 테스트)
- Apple Pay 추가 시 **+22.3% 전환율**, **+22.5% 매출** 증가
- 업계 평균 장바구니 이탈률 70% 대비 Stripe 사용자는 **10% 낮은 이탈률**

> **이박사 팩트 체크:** "위 전환율 데이터 출처는 Stripe 자체 블로그($1.4T 거래량 기반 A/B 테스트). Stripe가 자사 유리한 데이터만 공개했을 가능성 있음. 신뢰도: **중**. 독립적 제3자 검증 데이터는 부족."

> **박이사 결론:** "연마기 B2B에는 당장 Stripe가 필수는 아니다. 하지만 SaaS를 글로벌로 확장하려면 Stripe 없이는 사실상 불가능하다. 한국 PG로 해외 결제를 받으면 승인 거부가 속출한다. **SaaS가 준비되면 그때 Stripe를 붙여야 한다.**"

## 4. 최실장 실행 로드맵 — 4주 스케줄

### 전제 조건
대표님이 **Go** 결정을 내린 경우의 실행 스케줄.

| 주차 | 기간 | 작업 내용 | 담당 | 비용 |
|------|------|-----------|------|------|
| **1주차** | Day 1~7 | ① 미국 전화번호 확보 (Mint Mobile eSIM) ② 미국 가상 주소 설정 (Stable) ③ Stripe Atlas 신청서 제출 ($500 결제) ④ 한국 외국환은행에 해외직접투자 사전 신고 | 대표님 + 트럼프 | $500 + 전화/주소 셋업 |
| **2주차** | Day 8~14 | ① 법인 설립 완료 확인 (2영업일 내) ② Mercury Bank 계좌 신청 (EIN 발급 전 가능) ③ 한-미 세무 전문 회계사 선임 | 최실장 | 회계사 계약금 |
| **3주차** | Day 15~21 | ① EIN 발급 대기 (비미국인 15~30영업일) ② Mercury 계좌 승인 확인 ③ Stripe 결제 계정 설정 시작 | 개발팀 (강팀장) | - |
| **4주차** | Day 22~28 | ① EIN 수령 → Stripe 결제 계정 완전 활성화 ② 테스트 결제 진행 ③ SaaS 결제 페이지 프로토타입 연동 | 개발팀 (이대리) | - |

**주의:** EIN 발급이 15~30영업일이므로, 실제로는 **6~8주**가 소요될 수 있음. 4주 스케줄은 병렬 작업 기준이며 EIN은 별도 대기.

## 5. 트럼프·이박사 비판적 검증

### 트럼프 — 악마의 대변인 (Devil's Advocate)

**"영상에서 안 알려주는 것들을 까발리겠다."**

#### 숨은 비용 총정리 (LCC = Life Cycle Cost)

| 비용 항목 | 영상에서 언급? | 실제 금액 | 위험도 |
|-----------|--------------|-----------|--------|
| 법인 설립비 | O | $500 | 낮음 |
| 연간 Delaware 세금 | X | $300/년 (LLC) | 중간 |
| 등록 대리인 2년차~ | △ | $100/년 | 낮음 |
| Form 5472 미제출 벌금 | **X** | **건당 $25,000** | **치명적** |
| 한-미 양국 회계사 비용 | X | 400~800만원/년 | 높음 |
| 해외금융계좌 미신고 벌금 | **X** | **미신고 금액의 10~20%** | **치명적** |
| 해외직접투자 미신고 과태료 | **X** | **위반금액의 2% (최소 100만원)** | 높음 |
| Mercury 계좌 거부 리스크 | X | 시간 낭비 | 중간 |
| 법인 해산 비용 (사업 포기 시) | **X** | $200~$500 + 미납 세금 전액 | 중간 |

**5년 총 비용 시뮬레이션 (매출 $0인 최악의 경우):**

```
초기 설립:                $500
연간 유지비 × 5년:        $907 × 5 = $4,535
회계사 비용 (최소):       $1,500 × 5 = $7,500
해산 비용:                $500
────────────────────────────────────────
5년 총 비용:             약 $13,035 (약 1,825만원)
```

> **트럼프:** "매출 $0이어도 5년간 1,825만원이 증발한다. 이게 '위대한 비즈니스'인가? 아니다. 이건 돈을 태우는 거야. **매출이 먼저, 법인은 그 다음이다.** 순서를 바꾸면 안 된다."

#### 외환거래법 리스크 (이건 진짜 위험하다)

| 의무 | 미이행 시 제재 | 비고 |
|------|--------------|------|
| 해외직접투자 사전 신고 (외국환은행) | 위반금액의 2%, 최소 100만원 | 10억 초과 시 형사 고발 |
| 투자 후 사후보고 (6개월/연간) | 최대 1,400만원 과태료 | 반기/연간 보고 |
| 해외금융계좌 신고 (5억 초과 시) | 미신고 금액의 10~20%, 50억 초과 시 형사처벌 | 매년 6월 |
| CFC 과세 (국외지배회사) | 간주배당 과세 (한국 소득세율 적용) | LLC pass-through 구조가 오히려 독이 될 수 있음 |

> **트럼프:** "여기가 진짜 지뢰밭이다. Form 5472 하나 안 내면 $25,000 벌금, 해외금융계좌 안 신고하면 잔액의 20%. 한국 세무사 + 미국 CPA를 동시에 쓰지 않으면 반드시 사고 난다. **이 비용을 커버할 수 없으면 절대 시작하지 마라.**"

### 이박사 — 팩트 체크

| 주장 | 검증 결과 | 신뢰도 |
|------|-----------|--------|
| "Stripe Atlas $500이면 미국 법인 설립 가능" | **사실** — 다만 유지비 별도 | **상** |
| "EIN 2영업일 내 발급" | **조건부 사실** — SSN 보유자만. 한국인은 15~30영업일 | **중** |
| "Mercury Bank 쉽게 개설" | **주의 필요** — 2025년부터 비거주자 심사 대폭 강화, 거부 사례 증가 | **하** |
| "미국 법인이면 세금 절약" | **과장** — LLC pass-through로 미국 법인세 0%이나, 한국 CFC 과세로 상쇄 가능 | **하** |
| "Stripe 전환율 +8% 향상" | **Stripe 자체 데이터** — 독립 검증 부족, 이해충돌 가능성 | **중** |
| "한국 PG로 해외 결제 받기 어려움" | **사실** — 한국 PG는 해외 카드 승인 거부율 높음 | **상** |
| "Delaware가 최적의 주" | **논쟁 중** — Wyoming LLC가 연간 $200~300 저렴, 프라이버시도 우수 | **중** |
| "코드팩토리가 미국 법인 설립" | **확인 불가** — 코드팩토리(최지호)의 Stripe Atlas 경험 공개 자료 없음 | **하** |

> **이박사 결론:** "스트라이프 애틀라스 자체는 훌륭한 서비스다. 하지만 한국인 거주자에게 적용할 때의 세무·외환 리스크가 유튜브 영상에서는 거의 다뤄지지 않는다. **한-미 세무 전문가 상담 없이 진행하면 안 된다.** 데이터 신뢰도가 '하'인 항목이 3건 — 추가 검증이 필요하다."

## 7. Plan B — 전제가 틀렸을 경우의 대안

### 전제: "미국 법인이 글로벌 확장에 필수적이다"가 틀렸다면?

**Plan B-1: Stripe 없이 글로벌 SaaS 결제**
- **Paddle** (MoR 모델) — Paddle이 판매자 대리인으로 세금·결제를 모두 처리. 미국 법인 불필요
- **Lemonsqueezy** — 마찬가지로 MoR 모델, 인디 개발자에게 인기
- 수수료는 5~10%로 Stripe보다 높지만, 법인 설립/유지비 제로

**Plan B-2: 한국 법인으로 해외 결제 수취**
- **PortOne (구 아임포트)** — 한국 법인으로 해외 결제 연동 가능
- **PayPal Business** — 한국 사업자로 해외 결제 수취 가능 (수수료 4.4% + 고정)
- 단점: 결제 승인률 낮음, 지원 통화 제한

**Plan B-3: 단계적 접근 (추천)**
1. 먼저 Gumroad, Lemonsqueezy 등으로 SaaS 해외 판매 테스트
2. 월 매출 $5,000+ 달성 시 → 미국 LLC 설립 검토
3. 월 매출 $20,000+ 달성 시 → 본격 Stripe Atlas 진행
