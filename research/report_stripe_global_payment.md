## 1. 연마기 B2B 사업을 위한 Stripe 활용

### 1.1 국제 B2B 결제 지원

Stripe는 **46개국**에서 정식 서비스를 운영하며, **195개국 이상**의 고객으로부터 **135개 이상의 통화**로 결제를 받을 수 있다. 일본과 한국 모두 Stripe 지원 국가에 포함되어 있어, 한일 간 연마기 거래에 직접 활용 가능하다.

**B2B 결제 특화 기능:**
- **자동화 인보이스**: 브랜딩, 결제 조건, 세부 항목을 커스터마이징할 수 있는 인보이스 자동 생성 및 발송
- **인보이스 통합**: 여러 청구 건을 하나의 인보이스로 합산하여 정산 간소화
- **결제 조건 설정**: Net 30, Net 60 등 B2B 표준 결제 조건 설정 가능
- **자동 독촉 관리(Dunning)**: 미수금 자동 리마인더 및 재시도로 수금률 향상
- **스테이블코인 결제(2025년 신규)**: USDC 기반 결제 지원으로 국제 송금 2~5일 → 수분 내 정산

**연마기 B2B 거래에서의 실질적 이점:**
- 일본 바이어가 JPY로 결제 → 판매자는 USD로 정산 (자동 환전)
- 정기 소모품(연마석, 부품) 납품의 경우 구독/반복 결제 설정 가능
- 거래 내역이 대시보드에서 실시간 추적 가능

### 1.2 다중 통화 지원

| 항목 | 세부 내용 |
|------|-----------|
| 지원 통화 수 | 135개 이상 |
| 자동 환전 | 고객 통화 → 판매자 정산 통화 자동 변환 |
| 수수료 (국내) | 2.9% + $0.30 |
| 수수료 (해외) | 3.9% + $0.30 (해외카드 1% 추가) |
| 환전 추가 수수료 | 1% (통화 변환 시) |

### 1.3 미국 결제 게이트웨이의 B2B 신뢰도

미국 기반 결제 시스템을 사용하는 것은 국제 B2B 거래에서 중요한 신뢰 요소이다:

- **글로벌 표준**: Stripe는 연간 **$1.4조** (전 세계 GDP의 약 1.3%)를 처리하는 세계 최대급 결제 플랫폼
- **Fortune 500 기업 사용**: Shopify, DoorDash 등 세계적 기업이 Stripe을 사용
- **PCI DSS 준수**: 최고 수준의 보안 인증으로 기업 고객 안심
- **ISO 20022 지원**: 2027년까지 대부분의 국제 결제가 이 표준을 채택할 예정이며, Stripe가 선제 대응 중

## 2. SaaS 글로벌 진출을 위한 Stripe 기능

### 2.1 SaaS 빌링 모델 (Stripe Billing)

Stripe Billing은 모든 SaaS 가격 모델을 지원한다:

| 가격 모델 | 설명 | 예시 |
|-----------|------|------|
| **정액제 (Flat Rate)** | 월/연 고정 요금 | 월 $49 플랜 |
| **좌석 기반 (Per-Seat)** | 사용자 수에 따른 과금 | 사용자당 월 $10 |
| **사용량 기반 (Usage-Based)** | 실제 사용량에 따른 과금 | API 호출당 $0.01 |
| **단계별 (Tiered)** | 사용량 구간별 단가 변동 | 1~100건: $1, 101~1000건: $0.5 |
| **패키지 (Package)** | 묶음 단위 과금 | 1,000건 단위 $5 |
| **하이브리드** | 기본료 + 사용량 결합 | 월 $29 + API 초과 사용분 |

**고급 사용량 기반 빌링 (신규):**
- Stripe가 Metronome 인수 후 강화된 기능
- 수백만 건의 사용 이벤트 실시간 처리 가능
- 선불 크레딧 소진(burndown) 및 자동 크레딧 발급
- Rate Card 기반 미터링 항목별 가격 정의

**추가 핵심 기능:**
- **비례 계산(Proration)**: 플랜 변경 시 자동 비례 청구
- **맞춤 빌링 주기**: 일별, 주별, 월별, 거래별 과금 가능
- **실시간 청구 미리보기**: 빌링 주기 종료 전 예상 금액 확인
- **자동 매출 인식**: 사용량 변동에도 정확한 매출 보고
- **셀프서비스 고객 포털**: 고객이 직접 구독 관리 가능

### 2.2 Stripe의 글로벌 커버리지

| 항목 | 수치 |
|------|------|
| 정식 서비스 국가 | 46개국 |
| 확장 네트워크 (Paystack) | 5개국 (아프리카) |
| 프리뷰 시장 | 인도, 인도네시아 |
| 고객 결제 가능 국가 | 195개국 이상 |
| 지원 통화 | 135개 이상 |
| Stripe Tax 지원 국가 | 100개국 이상 |
| Stripe Tax 제품 유형 | 600개 이상 |
| Connect 글로벌 송금 | 118개국 이상 |

**지역별 지원 현황:**
- **북미/유럽**: 완전 지원 (미국, 캐나다, UK, 프랑스, 독일, 이탈리아 등)
- **아시아태평양**: 일본, 싱가포르, 호주, 뉴질랜드, 홍콩 등 지원
- **한국**: 한국 법인으로는 Stripe 계정 개설 불가 (미국 법인 필요)

### 2.3 Stripe Connect (마켓플레이스 모델)

SaaS가 마켓플레이스/플랫폼 모델로 확장할 경우:

- **3가지 온보딩 방식**: Stripe 호스팅, 임베디드(화이트라벨), API 기반
- **3가지 계정 유형**: Express, Custom, Standard
- **수익 분배**: 각 거래에서 플랫폼 수수료 자동 분배
- **에스크로 기능**: 조건 충족 시까지 대금 보류 가능
- **즉시 출금**: 판매자가 수익을 수분 내 은행 계좌로 출금 가능
- **글로벌 송금**: 118개국 이상으로 로컬 법인 없이 송금 가능
- **부가 금융 상품**: 브랜드 신용카드 발급(Issuing), 대출(Capital) 등 화이트라벨 제공

### 2.4 Stripe Tax (세금 자동화)

글로벌 SaaS 사업에서 가장 복잡한 문제 중 하나인 세금을 자동화:

- **100개국 이상** 자동 세금 계산 (판매세, VAT, GST)
- **600개 이상 제품 유형**에 대한 세금 분류
- **임계값 모니터링**: 각국 세금 등록 의무 자동 알림
- **B2B 세금 ID 검증**: 유럽/호주 VAT ID 자동 검증 및 역과세 적용
- **파일링 파트너 연동**: Taxually, Marosa, HOST와 자동 데이터 동기화
- **99.999% 업타임**: 실시간 세금 계산

**가격:**
- Tax Basic: 자동 계산만 (거래량 기반)
- Tax Complete: 등록 + 계산 + 파일링 통합 솔루션

## 3. 마케팅 관점의 Stripe 이점

### 3.1 고객 신뢰도

- Stripe는 연간 **$1.4조** 처리 (전 세계 GDP의 ~1.3%)
- 미국 기반 빌링은 글로벌 고객에게 **결제 보안과 신뢰도**를 제공
- PCI DSS Level 1 인증으로 최고 수준의 데이터 보안
- ML 기반 사기 탐지(Radar)가 수백만 기업의 데이터로 훈련

### 3.2 결제 전환율 (Conversion Rate)

핵심 통계:

| 지표 | 수치 |
|------|------|
| Apple Pay 도입 시 전환율 증가 | **+22.3%** |
| Apple Pay 도입 시 매출 증가 | **+22.5%** |
| WeChat Pay 도입 시 전환율 증가 | **+13%** |
| Stripe Checkout vs 타 솔루션 전환율 | **+8% 이상** (A/B 테스트 결과) |
| Express Checkout Element (Apple Pay 조기 노출) | **전환율 2배** |
| Stripe 사용 기업의 장바구니 이탈율 | **업계 평균 대비 10% 낮음** (Baymard Institute) |
| 온라인 장바구니 이탈율 (업계 평균) | **70%** |

**전환율 개선 요인:**
- 50개 이상의 글로벌 결제 수단을 동적으로 노출
- 실시간 카드 유효성 검증 (40% 미만의 사이트만 제공)
- 만료 카드 자동 차단 (30%의 사이트만 제공)
- 올바른 자동 완성 설정 (20%의 사이트가 미설정)

### 3.3 결제 수단 커버리지

**디지털 월렛:**
- Apple Pay
- Google Pay
- Cash App Pay
- WeChat Pay (9억+ 활성 사용자)
- Alipay

**지역별 결제 수단:**
- BLIK (폴란드 이커머스의 70%)
- iDEAL (네덜란드)
- Bancontact (벨기에)
- SEPA Direct Debit (유럽)
- Klarna (BNPL)
- Multibanco (포르투갈)
- 한국 로컬 카드 및 월렛 (해외 사업자용, 베타)

**기타:**
- 신용/체크카드 (Visa, Mastercard, AMEX, JCB, UnionPay)
- ACH 은행 이체
- 스테이블코인 (USDC)
- 암호화폐

### 3.4 개발자 생태계 및 통합

**API 및 개발 도구:**
- 통합 REST API (결제, 빌링, 구독, 송금, 금융 워크플로우)
- 7개 이상 프로그래밍 언어 SDK
- AI/LLM 통합 지원 및 MCP 서버 제공 (AI 에이전트가 Stripe와 상호작용)
- 80개 이상의 Partner Solution Blueprint

**주요 파트너 생태계:**
- 기술 파트너: AWS, Snowflake, MuleSoft, ServiceNow
- 이커머스: Shopify, WooCommerce
- 회계/ERP: Xero, DocuSign
- 컨설팅: Accenture, IBM, Slalom, WPP

**3줄 코드 연동:**
- 가입 → API 코드 복사 → 홈페이지에 삽입의 3단계로 결제 시스템 구축

## 4. 한국 PG vs Stripe 비교

### 4.1 구조적 차이

| 구분 | Stripe (글로벌 PG) | 한국 PG (NicePay, Toss, KG Inicis 등) |
|------|-------------------|--------------------------------------|
| **한국 법인 사용** | 불가 (해외 법인 필요) | 가능 |
| **글로벌 결제 승인율** | 높음 (각국 로컬 매입) | 낮음 (Cross-border 처리) |
| **기본 수수료** | 2.9% + $0.30 | 2.5~3.5% (국내카드) |
| **해외카드 수수료** | 3.9% + $0.30 | 3.8%~ |
| **SaaS 구독 빌링** | 우수 (모든 가격 모델) | 제한적 |
| **정산 통화** | USD 등 외화 | KRW (원화) |
| **지원 국가 (판매자)** | 46개국 | 한국만 |
| **지원 통화** | 135개+ | 제한적 |
| **세금 자동화** | 100개국+ 자동 | 없음 |
| **마켓플레이스 기능** | Connect (풀 기능) | 제한적 |
| **초기 진입 장벽** | 해외 법인 설립 필요 | 낮음 |

### 4.2 글로벌 승인율 문제

국내 PG사의 구조적 한계:
- 국내 PG사는 **국내 매입사를 경유**하여 Cross-border 거래를 처리
- 해외 발급사의 이상거래탐지시스템(FDS)이 **고위험군으로 분류**
- 결과적으로 **승인 거절이 빈번하게 발생**

Stripe의 이점:
- 전 세계 주요 거점에 **로컬 매입 라인** 구축
- 해외 거래도 **'로컬 거래'로 처리**되어 승인율 대폭 향상
- ML 기반 사기 탐지가 오탐(false positive)을 줄여 정상 거래 승인율 향상

### 4.3 한국 SaaS의 현실적 전략

한국 SaaS 스타트업이 글로벌 결제를 도입하는 단계적 전략:

1. **초기 단계**: MoR 서비스(Paddle, Lemon Squeezy) 활용 → 수수료 5~7%지만 법인 설립 불필요
2. **성장 단계**: Stripe Atlas로 미국 법인 설립 → Stripe 직접 사용 (2.9% + $0.30)
3. **확장 단계**: Stripe Connect로 마켓플레이스 모델 구축

## 5. Stripe Atlas를 통한 미국 법인 설립

### 5.1 개요

Stripe Atlas는 140개국 이상의 창업자가 미국 델라웨어주에 법인을 설립할 수 있는 원스톱 서비스이다.

### 5.2 비용 및 일정

| 항목 | 내용 |
|------|------|
| 설립 비용 | **$500** (법인 설립 + 첫해 등록 대리인) |
| 연간 유지 비용 | **$100** (등록 대리인 갱신) |
| 델라웨어 LLC 연간 세금 | **$300** (정액) |
| 법인 설립 소요 기간 | **약 2~3주** (100% 온라인) |
| EIN 발급 소요 기간 | **약 1주** |
| 법인 통장 개설 | **2영업일 내** (Mercury 은행) |

### 5.3 포함 서비스

- **법인 설립**: 델라웨어 C Corporation 또는 LLC
- **EIN 발급**: 미국 국세청(IRS) 세금 납부번호
- **법인 은행 계좌**: Mercury 은행 (EIN 발급 전에도 개설 가능)
- **법적 문서**: 정관, 운영 계약서, 주식 발행 서류
- **등록 대리인**: 연간 서비스
- **83(b) 세금 선택 신고**: 창업자 주식 관련
- **스타트업 혜택**: Stripe 제품 $2,500 크레딧 + $50,000 이상 파트너 할인 (AWS, Carta, Xero 등)

### 5.4 한국 창업자를 위한 고려사항

**장점:**
- SSN 없이, 미국 거주 없이 법인 설립 가능
- 델라웨어주는 Fortune 500 기업의 2/3가 선택한 주
- VC 투자에 유리한 법적 환경
- 미국 금융 시스템 접근으로 글로벌 신뢰도 확보

**주의사항:**
- 미국 법인세율 21% 적용
- 비거주 외국인은 이중과세 또는 30% 원천징수 가능 (한미 조세 조약 확인 필요)
- 미국 세금 시스템이 복잡하여 세무 전문가 필수
- 한국 해외직접투자 규정 확인 필요

## 6. 코드팩토리 참고 정보

### 6.1 코드팩토리 (CodeFactory) 소개

코드팩토리(최지호)는 한국의 개발자 교육 플랫폼으로, 주로 **Dart/Flutter** 관련 강의를 제공한다.

- **배경**: 임페리얼 칼리지 런던 졸업, 밀리만 한국 지사 소프트웨어 엔지니어 출신
- **활동**: 인프런, 패스트캠퍼스 등에서 Flutter 강의 제공
- **실적**: 밀리의서재 Flutter 전환 프로젝트 리드
- **GitHub**: 56개 리포지토리 보유 (github.com/codefactory-co)
- **웹사이트**: codefactory.ai

### 6.2 Stripe Atlas 관련

코드팩토리의 Stripe Atlas를 통한 미국 법인 설립 경험에 대한 **직접적인 공개 자료는 확인되지 않았다**. 다만, 한국 개발자/SaaS 사업자 커뮤니티에서 Stripe Atlas를 활용한 미국 법인 설립 사례는 다수 존재한다:

- Medium의 이은강 님 시리즈: "Stripe Atlas로 미국에 스타트업 세우기" (1~3편)
- 클리앙: "70만원으로 미국 법인 세우기 - Stripe Atlas"
- 아산 보이저: "Stripe 서비스 통해 미국 법인 설립하기"

코드팩토리의 구체적인 경험은 해당 채널(YouTube, Discord 등)에서 직접 확인이 필요하다.

## 7. 핵심 수치 요약

### Stripe 플랫폼 규모
- 연간 처리 금액: **$1.4조** (전 세계 GDP의 ~1.3%)
- 서비스 국가: **46개국** (정식) + **195개국** (결제 수취)
- 지원 통화: **135개 이상**
- 글로벌 B2B 결제 시장 규모: **$97.88조** (2025년 추정)

### 전환율 개선 효과
- Apple Pay 도입: 전환율 **+22.3%**, 매출 **+22.5%**
- WeChat Pay 도입: 전환율 **+13%**, 매출 **+14%**
- Stripe Checkout: 타 솔루션 대비 전환율 **+8%**
- Express Checkout: 전환율 **2배**
- 장바구니 이탈율: 업계 평균 대비 **10% 낮음**

### 비용 구조
- 국내 카드: **2.9% + $0.30**
- 해외 카드: **3.9% + $0.30**
- Stripe Atlas 법인 설립: **$500** (1회) + **$100/년**
- 델라웨어 LLC 세금: **$300/년**

### SaaS 빌링
- 지원 가격 모델: **7가지 이상** (정액, 좌석, 사용량, 단계별, 패키지, 볼륨, 변동)
- Stripe Tax: **100개국+ 자동 세금 계산**
- Connect 글로벌 송금: **118개국+**
