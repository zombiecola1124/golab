## 2. ★ 핵심 테이블: 리베이트 역산표

> "내가 매달 현금 리베이트 X만 원 줘야 하면, 연봉을 얼마로 잡아야 하나?"

### 방법 A: 전액 내 월급에서 (순수 개인 돈)

```
생활비를 월 400만 원 쓴다고 가정:

┌──────────┬──────────┬──────────┬──────────┬──────────┬──────────┐
│ 리베이트  │ 필요     │ 연봉     │ 회사부담  │ 법인에    │ 법인세   │
│ (월)     │ 실수령/월 │ 세팅     │ 보험 포함 │ 남는돈   │ (9%)    │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
│ 100만    │  500만   │ 7,500만  │ 8,138만  │ 4,062만  │  366만  │
│ 150만    │  550만   │ 8,000만  │ 8,680만  │ 3,520만  │  317만  │
│ 200만    │  600만   │ 9,000만  │ 9,765만  │ 2,435만  │  219만  │
│ 250만    │  650만   │ 9,800만  │ 10,633만 │ 1,567만  │  141만  │
│ 300만    │  700만   │ 1.05억   │ 11,393만 │   807만  │   73만  │
│ 350만    │  750만   │ 1.12억   │ 12,152만 │    48만  │    4만  │
│ 400만    │  800만   │ 1.20억   │ 13,020만 │  ▲적자   │    0   │
└──────────┴──────────┴──────────┴──────────┴──────────┴──────────┘

▲ 리베이트 월 400만이면 이익 1.5억으로는 급여를 못 감당
   → 방법 B(하이브리드) 필수
```

### 방법 B: 하이브리드 (판매장려금 + 내 돈)

```
판매장려금: 리베이트의 절반을 법인 비용(계좌이체)으로 처리
나머지 절반: 내 월급에서 현금

생활비 월 400만 가정:

┌──────────┬──────────┬──────────┬──────────┬──────────┬──────────┬──────────┐
│ 총       │ 장려금   │ 현금     │ 연봉     │ 회사     │ 법인     │ 법인세   │
│ 리베이트  │(법인비용) │(내 돈)   │ 세팅     │ 총비용   │ 남는돈   │ (9%)    │
│ (월)     │ (월)    │ (월)     │         │         │         │         │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
│ 200만    │ 100만   │ 100만   │ 7,500만  │ 9,338만  │ 2,862만  │  258만  │
│ 300만    │ 150만   │ 150만   │ 8,000만  │ 10,480만 │ 1,720만  │  155만  │
│ 400만    │ 200만   │ 200만   │ 9,000만  │ 12,165만 │    35만  │    3만  │
│ 500만    │ 250만   │ 250만   │ 9,800만  │ 13,633만 │  ▲적자   │    0   │
└──────────┴──────────┴──────────┴──────────┴──────────┴──────────┴──────────┘

※ 회사 총비용 = 연봉 + 회사부담보험(~8.5%) + 판매장려금(연) + 유지비(1,800만)
※ 판매장려금은 법인 비용처리 → 법인세 절감 효과
※ 장려금 부분은 거래처 사업자 통장으로 계좌이체 (약정서 필요)
```

## 3. 한 눈에 보기: 추천 시나리오 3개

### 시나리오 ①: 리베이트 적음 (월 100~150만)

```
연봉: 8,000만
월 실수령: 542만
생활비: 400만
현금 리베이트: 142만 (전부 내 돈)
법인 남는 돈: 3,520만 → 법인세 317만
총 세부담: 소득세 851만 + 법인세 317만 + 보험 1,280만 = 2,448만

→ 리베이트 규모가 작으면 이게 제일 간단
```

### 시나리오 ②: 리베이트 중간 (월 200~300만) ★추천

```
연봉: 9,000만
판매장려금(법인 비용): 월 150만 (연 1,800만)
월 실수령: 604만
생활비: 400만
현금 리베이트: 150만 (내 돈) + 150만 (법인 장려금) = 총 300만
나머지: 604 - 400 - 150 = 54만 여유
법인 남는 돈: 150M - 97.65M(연봉+보험) - 18M(장려금) - 18M(유지비) = 16.35M
법인세: 16.35M x 9% = 147만
총 세부담: 소득세 1,078만 + 법인세 147만 + 보험 1,360만 = 2,585만

→ 월 300만 리베이트 가능하면서 생활비도 확보
→ 판매장려금으로 절반을 법인비용 처리해서 절세
```

### 시나리오 ③: 리베이트 많음 (월 350만 이상)

```
연봉: 1.05억
판매장려금(법인 비용): 월 200만 (연 2,400만)
월 실수령: 690만
생활비: 400만
현금 리베이트: 200만 (내 돈) + 200만 (법인 장려금) = 총 400만
나머지: 690 - 400 - 200 = 90만 여유
법인 남는 돈: 150M - 113.9M(연봉+보험) - 24M(장려금) - 18M(유지비) = ▲5.9M 적자

⚠️ 적자 발생 → 이익 1.5억으로는 힘듦
→ 이익이 2억 이상이어야 가능
→ 또는 판매장려금 비율을 더 높이고 연봉을 낮추는 방법

대안: 연봉 8,000만 + 판매장려금 월 250만(연 3,000만)
→ 총비용: 86.8M + 30M + 18M = 134.8M
→ 남는돈: 15.2M → 법인세 137만
→ 실수령 542만 - 생활비 400만 = 현금 142만
→ 총 리베이트: 142만(현금) + 250만(장려금) = 392만
→ ★ 가능! 단, 장려금 250만이 계좌이체라서 거래처가 수용해야 함
```
