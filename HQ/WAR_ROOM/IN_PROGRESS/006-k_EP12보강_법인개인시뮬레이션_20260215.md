## 2. 사회적 이득 비교

## 3. 리베이트 시뮬레이션 (핵심!)

### 3-1. 개인사업자일 때 (현재)

```
현재 관행:
├── 거래처에 현금 리베이트(판촉비/장려금) 지급
├── 사업용 계좌에서 인출하여 지급
├── 또는 매출 일부를 현금으로 빼서 지급
└── 접대비 또는 판매장려금으로 비용처리 시도

세무 처리:
├── 적격증빙(세금계산서, 카드, 현금영수증) 있으면 → 비용 인정
├── 현금 지급 + 영수증/약정서 있으면 → 판매장려금 비용 인정 가능
└── 증빙 없는 현금 → 비용 불인정 + 세무조사 리스크

현실:
├── 개인사업자는 사업자금 ↔ 개인자금 경계 모호
├── 현금 인출/지급의 "자유도"가 높음
└── 다만 세무조사 시 소명 못하면 인정상여 처분 + 추징
```

### 3-2. 법인에서의 리베이트 — 5가지 적법한 방법

#### 방법 1: 판매장려금 (가장 추천!)

```
판매장려금(판매촉진비)
├── 개념: 매출 실적에 비례하여 거래처에 지급하는 장려금
├── 손금 한도: ★ 한도 없음 ★ (접대비와 달리!)
├── 부가세: 금전 지급 시 비과세 (세금계산서 불요)
├── 원천징수: 사업자에게 지급 시 의무 없음
│
├── 필수 증빙:
│   ├── ① 판매장려금 지급 약정서/계약서 (거래처별)
│   ├── ② 매출 실적 근거 자료
│   ├── ③ 계좌이체 내역 (법인→거래처 사업자 계좌)
│   └── ④ 수령확인서
│
└── 주의: 특정 1곳에만 편중 시 접대비로 재분류 위험
```

> **이것이 "현금 봉투" → "투명한 제도"로 바꾸는 핵심 방법입니다.**
> 실질은 같지만 형식을 갖추면 한도 없이 비용 인정됩니다.

#### 방법 2: 매출에누리 (부가세까지 줄일 수 있음)

```
매출에누리(매출할인)
├── 개념: 판매가격 자체를 할인하여 매출 감소
├── 손금 한도: 한도 없음
├── 부가세: ★ 공급가에서 제외 → 부가세 부담도 감소 ★ (가장 유리)
│
├── 필수 증빙:
│   ├── 수정세금계산서 발행 또는
│   └── 계약서상 할인 조건 사전 명시
│
└── 주의: 판매 시점에 가격 조정이 이루어졌는지 요건 엄격
```

#### 방법 3: 접대비

```
접대비(기업업무추진비)
├── 개념: 거래처 관계 유지 비용
├── 손금 한도: 매출 300억 기준 약 1억 600만 원
│   ├── 기본한도: 3,600만 (중소기업)
│   ├── 100억 x 0.3% = 3,000만
│   └── (300억-100억) x 0.2% = 4,000만
│
├── 증빙: 3만 원 초과 시 적격증빙 필수
├── 부가세: 매입세액 불공제
└── 용도: 식대, 선물 등 소액 관계 유지에 적합
```

#### 방법 4: 지급명세서 방식 (비추천)

```
거래처 종업원 개인에게 직접 리베이트 지급
├── 기타소득(사례금) → 원천징수 22% 의무 발생
├── 지급명세서 제출 필수
└── ⚠️ 과세관청 주목 대상 → 비추천
```

#### 방법 5: 매입리베이트 (받는 쪽)

```
공급사로부터 받는 리베이트
├── 매입원가에서 차감 처리
└── 영업외수익으로 처리하지 않도록 주의
```

### 3-4. 현금 리베이트 → 법인 전환 후 프로세스

```

  ↓ 증빙 없음, 리스크 높음

[법인 전환 후: 투명화]

  Step 1. 판매장려금 지급 규정 수립
  ├── 내부 규정 / 이사회 결의
  ├── 매출액 대비 ○% 또는 수량 기준 장려금 지급 기준
  └── 대상 거래처 범위 설정 (1곳만은 위험)

  Step 2. 거래처별 약정서 체결
  ├── 지급 조건, 지급 시기, 산출 근거 명시
  └── 양측 서명/날인

  Step 3. 계좌이체로 지급 (★ 현금 지급 최소화)
  ├── 법인 계좌 → 거래처 사업자 계좌
  └── 이체 내역 자체가 증빙

  Step 4. 수령확인서 징수
  └── 거래처로부터 판매장려금 수령확인서

  Step 5. 회계 처리
  ├── 판매촉진비(판매장려금) 계정
  ├── 부가세 비과세 → 세금계산서 불요
  └── 한도 없이 전액 비용 인정!
```
