## 1. 전문용어 쉬운 설명

### 판매장려금이란?

```
쉽게 말하면:
"너네가 우리 물건 많이 사줬으니까, 고마움의 표시로 돈 좀 줄게"

정식 이름: 판매장려금, 판매촉진비, 리베이트
= 거래처한테 "잘해줘서 고맙다"고 주는 돈

예시:
├── "이번 달 1톤 넘게 사줬으니 톤당 5만 원 돌려줄게" → 판매장려금
├── "올해 거래 많이 해줘서 연말에 좀 챙겨줄게" → 판매장려금
└── "다음에도 우리한테 사라"는 의미로 주는 돈 → 판매장려금
```

### 매출에누리란?

```
쉽게 말하면:
"처음부터 깎아주는 것"

예시:
├── "원래 키로당 100만 원인데, 너네는 95만 원에 줄게" → 매출에누리
├── 세금계산서를 95만 원으로 끊음
└── 처음부터 싸게 파는 거라서 나중에 돈 줄 필요 없음
```

### 접대비란?

```
쉽게 말하면:
"거래처 사장님 밥 사주고, 선물 주고, 술 사주는 돈"

예시:
├── 거래처 사장님 저녁 → 접대비
├── 명절 선물세트 → 접대비
└── 골프 → 접대비

한도: 매출 300억이면 연간 약 1억 600만 원까지만 비용으로 인정
```

### 가지급금이란?

```
쉽게 말하면:
"회사 돈을 사장이 먼저 가져간 것 — 아직 정산 안 한 상태"

회사 통장에서 돈을 빼면 → 장부에 "사장한테 빌려줬다" 기록됨
이게 "가지급금"

문제:
├── 세무서가 "이자 안 받았네?" → 연 4.6% 이자를 받은 것으로 간주 → 세금
├── 오래 안 갚으면 → "사장이 횡령한 거 아니야?" → 형사 문제
└── 1인 법인이라도 회사 ≠ 사장 → 회사 돈 마음대로 못 씀
```

### 인정상여란?

```
쉽게 말하면:
"세무서가 '이거 사장 월급이었잖아' 하고 세금 때리는 것"

회사 돈 빼서 쓴 내역을 소명 못하면
→ 세무서: "어 이거 사장 상여금(보너스)이었네?"
→ 소득세 추가로 내라!
→ 이게 "인정상여"
```

### 포괄양수도란?

```
쉽게 말하면:
"개인사업 통째로 법인한테 넘기는 것"

개인사업의 모든 것 (거래처, 재고, 장비, 계약 등)을
→ 새로 만든 법인에게 한방에 넘김
→ 부가세가 안 붙음 (이게 핵심!)
→ 이때 "영업권"도 같이 넘기면 절세 효과 큼
```

### 영업권이란?

```
쉽게 말하면:
"내가 쌓아온 장사 능력의 값어치"

거래처 네트워크, 노하우, 브랜드 인지도, 단골 등
→ 돈으로 환산하면 이게 "영업권"
→ 법인이 이걸 사면 → 5년에 걸쳐 비용처리 가능 → 법인세 절감
```

## 2. 현금 리베이트 — 현실적 방법 3가지

### 사장님 상황 정리

```
불변 조건: 거래처한테 현금으로 리베이트를 줘야 한다
이유: 업계 관행, 거래 유지, 관계 관리
방식: 현금 (계좌이체가 아닌 실물 현금)
```

### 방법 A: 월급에서 주기 (가장 안전)

```
구조:
  법인 → (월급) → 사장님 개인 통장 → (현금 인출) → 거래처에 전달

장점:
├── ★ 100% 합법 ★
├── 월급은 이미 소득세 + 4대보험 다 뗀 "내 돈"
├── 내 돈을 어디에 쓰든 아무도 뭐라 못 함
├── 세무서가 뭐라 할 근거 없음
└── 가장 깔끔하고 안전한 방법

단점:
├── 리베이트 금액만큼 법인 비용처리가 안 됨
│   └── 회사 입장에서는 "비용 절감" 효과가 없다는 뜻
├── 내 주머니에서 나가는 거라 실수령이 줄어듦
└── 리베이트 규모가 크면 월급을 많이 잡아야 함

예시 (월 리베이트 300만 원 필요 시):
├── 대표 연봉을 1.3~1.5억으로 설정
├── 세금 + 보험 뗀 후 월 실수령 약 850~950만 원
├── 여기서 300만 원을 현금으로 인출 → 리베이트
└── 남은 550~650만 원이 사장님 생활비
```

### 방법 B: 판매장려금(회사 비용) + 현금 병행

```
구조:
  일부: 법인 → (판매장려금, 계좌이체) → 거래처 사업자 통장
  일부: 사장님 월급 → 현금 → 거래처 담당자 개인

장점:
├── 계좌이체 부분은 법인 비용처리 가능 → 법인세 절감
├── 현금 부분은 사장님 개인 돈 → 합법
├── 거래처도 "공식 장려금 + 개인적 감사" 두 트랙으로 받음
└── 가장 현실적인 하이브리드 방법

실무:
├── 거래처와 판매장려금 약정서 체결 (공식 부분)
│   ├── "월 매입액의 2%를 장려금으로 지급"
│   ├── 법인 통장 → 거래처 사업자 통장으로 이체
│   └── 이 금액은 전액 법인 비용 인정 (한도 없음)
│
└── 추가로 사장님이 개인적으로 챙기는 부분 (비공식)
    ├── 사장님 월급에서 현금 인출
    ├── 개인 대 개인으로 전달
    └── 법인 장부와 무관 → 문제 없음
```

### 방법 C: 급여를 넉넉히 잡고 전부 개인 돈으로 처리

```
구조:
  법인 → (넉넉한 월급) → 사장님 → 현금 → 리베이트 전액

장점:
├── 가장 단순
├── 법인 장부에 리베이트 흔적 자체가 없음
├── 세무 리스크 0%
└── "내 월급이니까 내 마음"

단점:
├── 법인 비용처리 효과 없음
├── 월급을 높이면 소득세 + 4대보험이 올라감
└── 리베이트 규모가 매우 크면 월급도 매우 커야 함

예시 (연간 리베이트 3,000만 원 필요 시):
├── 대표 연봉 1.5억 설정 (EP-12보강 시나리오 C)
├── 세금+보험 후 실수령 약 1.04억
├── 리베이트 3,000만 원 차감
├── 사장님 실제 생활비: 약 7,400만 원
└── 법인에 남는 돈: 거의 0 → 법인세 0
```

## 4. 구체적 숫자 시뮬레이션

### 가정: 월 현금 리베이트 300만 원 필요

#### 방법 B (하이브리드) 적용

```
판매장려금 (법인 비용): 월 150만 → 연 1,800만
  ├── 거래처 사업자 통장으로 계좌이체
  ├── 법인 비용처리 → 법인세 절감 (1,800만 x 9% = 162만 원 절감)
  └── 약정서 + 이체 내역 = 증빙 완비

사장님 월급에서 현금: 월 150만 → 연 1,800만
  ├── 사장님 개인 통장에서 현금 인출
  ├── 개인 대 개인 전달
  └── 사장님 개인 돈 → 합법, 세무 리스크 0

대표 연봉 설정: 1.1억
  ├── 세금+보험 후 실수령: 약 월 650만
  ├── 현금 리베이트 차감: -150만
  ├── 사장님 실제 생활비: 월 약 500만
  └── 법인 잔여이익: 1.5억 - 1.1억 - 회사부담보험(740만) - 장려금(1,800만) = 약 1,260만
     └── 법인세 (9%): 약 113만 원
```

#### 전체 세부담 비교

| 항목 | 개인사업자 현재 | 법인(하이브리드) |
|------|--------------|----------------|
| 소득세+지방세 | 4,077만 | ~1,800만 (급여분) |
| 법인세 | - | ~113만 |
| 4대보험 본인부담 | 1,948만 | ~900만 |
| 법인 유지비 | 0 | 1,800만 |
| 성실신고 | 200만 | 0 |
| **총 부담** | **6,225만** | **~4,613만** |
| 판매장려금 (법인 비용) | 비용처리 안됨 | 1,800만 비용처리 |
| 현금 리베이트 (개인) | 내 돈에서 | 내 월급에서 |
| **실수령 (리베이트 제외)** | **~7,000만** | **~7,087만** |

> **트럼프 이사:** "숫자가 나왔습니다.
> 리베이트 포함해서 계산하면, 법인이 실수령도 비슷하면서 리스크 관리까지 됩니다.
> 판매장려금으로 절반을 법인 비용으로 넣으면 법인세도 줄고, 나머지는 사장님 돈으로 깔끔."

```
│                                                                 │
│  ✅ 대신 이렇게:                                                 │
│  ├── 월급으로 받은 후 → 내 돈으로 현금 주기 (합법)                │
│  └── 회사에서 판매장려금으로 계좌이체 (합법)                      │
│                                                                 │
│  핵심: "회사 돈" → "내 돈" 변환 절차를 거쳐야 함                  │
│       (월급, 배당, 경비 정산 등)                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 6. 정리: 사장님이 기억할 것 3가지

```
1. 현금 리베이트 = 내 월급에서 주면 100% 합법, 아무 문제 없음
2. 추가로 판매장려금 제도 만들면 → 법인 비용으로도 줄 수 있음 (절세)
3. 회사 돈 직접 빼서 주면 안 됨 → 월급이든 배당이든 "내 돈"으로 바꾼 후에

→ 방법 B (하이브리드)가 최적: 일부 회사 비용 + 일부 내 돈
```

## 7. 더쎈카드 기능 → ERP 연동 판단

```
결론: ❌ 연동 안 함 (별도 관리)

이유:
├── 카드 비교/추천은 카드고릴라, 뱅크샐러드가 이미 잘 하고 있음
├── ERP는 재고/매입/정산에 집중해야 함
├── 카드 기능 붙이면 ERP가 복잡해지고 유지보수 부담
└── 나중에 필요하면 v3에서 "사업비 지출 분석" 정도로 가볍게 추가

사장님이 하실 것:
├── 카드고릴라 또는 뱅크샐러드 앱 설치
├── 보유 카드 등록 → 최적 카드 자동 추천 받기
└── ERP와는 별개로 관리
```
