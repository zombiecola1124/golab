## 1. 사장님 브리핑

- 직원 고용한다면 법인에서 하는 게 좋은지, 개인에서 하는 게 좋은지
- 뭐가 나에게 이득이 될까
- 이것저것 나에게 도움이 될 방법들 전부 고려
- 현재: IRP, 연금저축펀드, ISA, 노란우산공제 하고 있음

## 2. 직원 고용: 법인 vs 개인사업자 비교

### 2-1. 4대보험 사업주 부담금 (2026년 기준)

| 보험 | 총 요율 | 사업주 부담 | 근로자 부담 | 비고 |
|------|---------|------------|------------|------|
| 국민연금 | 9.5% | 4.75% | 4.75% | 2026년 인상 (9%→9.5%) |
| 건강보험 | 7.19% | 3.595% | 3.595% | 2026년 인상 |
| 장기요양 | 건보의 0.9448% | 50% | 50% | 2026년 인상 |
| 고용보험(실업) | 1.8% | 0.9% | 0.9% | 동결 |
| 고용보험(안정/능력) | 0.25~0.85% | 전액 사업주 | - | 규모별 차등 |
| 산재보험 | 업종별 | 전액 사업주 | - | 도소매 약 0.7% |

### 2-2. 핵심 차이: 대표자 본인의 보험료 부담

| 구분 | 개인사업자 대표 | 법인 대표이사 |
|------|----------------|--------------|
| 보험료 산정 기준 | **종합소득 전체** | **대표이사 급여만** |
| 건강보험 | 소득+재산 기준 (지역) | 급여 기준 50% (직장) |
| 국민연금 | 소득금액 기준 | 급여 기준 50% |
| **절감 여지** | 거의 없음 | **급여 설정으로 조절 가능** |

> **트럼프 이사:** "288억 매출에서 순이익이 상당하면, 개인사업자 건보료+국민연금이 연 수천만 원입니다.
> 법인 전환 후 대표 급여 1억으로 설정하면 보험료만 연 수천만 원 절감."

### 2-3. 인건비 비용처리

| 항목 | 개인사업자 | 법인 |
|------|----------|------|
| **대표 본인 급여** | **비용처리 불가** ← 핵심 | 급여로 비용처리 가능 |
| 직원 인건비 | 비용처리 가능 | 비용처리 가능 |
| 퇴직급여 충당금 | 제한적 | 정관 규정에 따라 가능 |
| 4대보험 사업주 분 | 비용처리 가능 | 비용처리 가능 |

### 2-4. 퇴직금 비교

| 항목 | 개인사업자 | 법인 |
|------|----------|------|
| 직원 퇴직금 | 1년+ 근속 의무 지급 | 동일 |
| **대표 퇴직금** | **없음** | 정관 규정 시 지급 가능 |
| 임원 퇴직금 한도 | 해당 없음 | 퇴직 전 1년 총급여 x 1/10 x 근속연수 |
| 퇴직소득세 | - | **연분연승 분리과세 (세부담 매우 낮음)** |

### 2-5. 고용 형태별 비교

| 형태 | 사업주 부담 | 4대보험 | 퇴직금 | 해고 유연성 | 리스크 |
|------|------------|---------|--------|-----------|--------|
| **정규직** | 가장 높음 | 분담 | 의무 | 낮음 | 해고 제한 |
| **계약직** | 높음 | 분담 | 의무 | 중간 | 2년 초과 시 무기계약 전환 |
| **일용직** | 중간 | 조건부 | 없음 | 높음 | 근로자성 다툼 |
| **프리랜서(3.3%)** | 가장 낮음 | 없음 | 없음 | 매우 높음 | **2026년 가짜 3.3 단속 강화** ⚠️ |

> **⚠️ 2026년 핵심 변화:** 고용노동부가 '가짜 3.3% 계약' 집중 단속 예고.
> 실질적으로 사용자 지휘/감독 아래 근로 제공 시 → **근로자**로 판정
> → 4대보험 소급 가입 + 퇴직금 지급 책임 발생

### 2-6. 정부 지원금

**두루누리 사회보험료 지원:**
```
대상: 근로자 10인 미만 사업장, 월 임금 270만 원 미만 신규가입자
지원: 고용보험/국민연금 보험료의 80%를 최대 36개월 지원
주의: 사업주 종합소득 4,300만 원 이상 시 사업주 지원 제외
      (근로자 본인 지원은 유지)
```

**고용촉진장려금:** 취업취약계층(장기실업자, 고령자, 장애인 등) 고용 시 장려금

### 2-7. 가족 직원 고용

**장점:**
- 실제 근무 시 인건비 **전액** 비용처리 가능
- 소득 분산 효과 → 누진세율 완화
- 동일 세대원 친족은 **최저임금 적용 제외** 가능

**세무 리스크:**
| 위험 | 내용 |
|------|------|
| 허위 고용 | 실제 근무 없이 인건비 신고 시 **탈세** → 추징 + 가산세 |
| 과다 급여 | 동일 직급 대비 과도한 급여 → 초과분 불인정 |
| 증빙 부족 | 근로계약서, 출퇴근 기록, 계좌이체 없으면 소명 곤란 |

**배우자 고용 특수사항:**
- 국민연금/건강보험: 가입 가능
- **고용보험/산재보험: 가입 불가** (배우자인 경우)

**필수 증빙:** 근로계약서 + 계좌이체 급여 지급 + 출퇴근 기록부 + 원천세 매월 신고

## 3. 현재 절세/재테크 최적화 (이미 하고 있는 것)

### 3-1. 노란우산공제

| 항목 | 내용 |
|------|------|
| 납입 한도 | 월 5만~100만 원 (연간 최대 **1,200만 원**) |
| 소득공제 (사업소득 1억 초과) | **200만 원** |
| 절세 효과 (45% 세율 기준) | 약 **90만 원/년** |
| 법인대표 적용 | 총급여 8천만 원 이하 시 가능 |

> 288억 매출 → 소득 1억 초과 → 공제 한도 겨우 200만 원. 단독으로는 효과 미미.
> 다만 폐업/퇴직 시 안전망 역할 + 복리 수익이 있으므로 유지 권장.

### 3-2. IRP (개인형 퇴직연금)

| 항목 | 내용 |
|------|------|
| 납입 한도 | 연금저축 + IRP 합산 연 **1,800만 원** |
| 세액공제 한도 | 연금저축 포함 **900만 원** |
| 세액공제율 (고소득) | **13.2%** |
| 최대 세액공제 | **118.8만 원** (900만 x 13.2%) |

### 3-3. 연금저축펀드

| 항목 | 내용 |
|------|------|
| 세액공제 한도 | 연 **600만 원** |
| IRP와 합산 | 900만 원 한도 |
| 수령 시 세율 | 연금소득세 3.3~5.5% (vs 소득세 45%) |

### 3-4. ISA (개인종합자산관리계좌)

| 항목 | ~2025년 | **2026년~** |
|------|---------|-----------|
| 연간 납입한도 | 2,000만 원 | **4,000만 원** |
| 총 누적 한도 | 1억 원 | **2억 원** |
| 비과세 한도(일반) | 200만 원 | **500만 원** |
| 초과 수익 과세 | 9.9% 분리과세 | 9.9% 분리과세 |

**핵심 전략 — ISA → 연금 전환:**
- ISA 만기 자금을 연금저축/IRP로 이체 시
- 이체금액의 10% (최대 300만 원) **추가 세액공제**
- 기본 900만 원 + ISA 전환 300만 원 = **최대 1,200만 원** 세액공제

### 3-5. 종합 절세 효과

| 상품 | 공제 유형 | 연간 한도 | 최대 절세 |
|------|----------|----------|----------|
| 노란우산 | 소득공제 | 200만 | ~90만 원 |
| 연금저축+IRP | 세액공제 | 900만 | ~118.8만 원 |
| ISA→연금 전환 | 추가 세액공제 | 300만 | ~39.6만 원 |
| **합계** | | | **연간 약 210~250만 원** |

> **트럼프 이사:** "솔직히 말합니다. 288억 매출 사장님에게 연간 250만 원 절세는 티끌입니다.
> **법인 전환 = 연간 수천만~1억 원 절세.** 게임 자체가 다릅니다."

## 4. 추가 절세/재테크 방법

### 4-1. 중소기업 특별세액감면

```
적용 업종: 도소매업 포함
적용 기한: 2028년 12월 31일까지

288억 매출 → 소기업(50억 이하) 기준 초과 → "중기업"
├── 수도권 사업장: 감면 불가
└── 수도권 외(지방): 5% 감면 (한도 1억 원)

→ 지방 사업장이면 활용, 수도권이면 해당 없음
→ 신청주의! 세액감면 신청서 직접 제출해야 함
```

### 4-2. 업무용 차량 비용처리

| 항목 | 한도 |
|------|------|
| 감가상각비 | 연 **800만 원** (5년 정액법) |
| 유지관리비 | 연 **700만 원** |
| 차량 1대 합계 | 연 최대 **1,500만 원** |
| 운행일지 작성 시 | 업무사용 비율만큼 추가 인정 |

> **⚠️ 2026년 변화:** 업무용 승용차 2대+ 보유 시, 1대 제외 나머지는
> **업무전용 자동차보험** 필수 가입. 미가입 시 **비용 전액 불인정**.

### 4-3. 해외출장비 처리 (일본 전시회 등)

```
비용처리 가능:
├── 항공료, 숙박비, 현지 교통비, 식대/일비
├── 전시회 참가비/등록비
├── 비자/보험료, 통역비
└── 출장 보고서, 전시회 참가 확인서 등 증빙 필수

필수 증빙:
├── 출장 목적서/계획서 (사업 관련성 입증)
├── 전시회 참가 확인서, 입장 뱃지
├── 항공권/e-티켓, 숙박/교통 영수증
├── 출장 보고서 (업무 결과 기록)
└── 수집한 명함/카탈로그 (보조 증빙)

주의: 관광 일정과 명확히 구분! 혼재 시 전액 부인 위험
```

### 4-4. 접대비 한도 (288억 매출 기준)

```
기본한도 (중소기업): 3,600만 원
+ 100억 x 0.3%     = 3,000만 원
+ (288억-100억) x 0.2% = 3,760만 원
━━━━━━━━━━━━━━━━━━━━━━
합계: 약 1억 360만 원

증빙: 건당 3만 원 초과 시 적격증빙(카드/세금계산서/현금영수증) 필수
```

### 4-5. 건강보험료/국민연금 최적화

```
개인사업자의 문제: 종합소득 전체가 보험료 산정 기준
→ 사실상 최적화 방법 제한적

유일한 해결책: 법인 전환
→ 대표이사 급여에 대해서만 보험료 부과
→ 급여 월 500~900만 원 설정 시 보험료 대폭 절감
```

## 5. 법인 전환 시 종합 시너지 (핵심 권고)

### 5-1. 세율 비교

```
개인사업자 소득세율:
├── 5,000만~8,800만: 24% (지방세 포함 26.4%)
├── 8,800만~1.5억: 35% (38.5%)
├── 1.5억~3억: 38% (41.8%)
├── 3억~5억: 40% (44%)
├── 5억~10억: 42% (46.2%)
└── 10억 초과: 45% (49.5%)  ← 사장님 현재 적용 구간

법인세율:
├── 2억 이하: 9% (9.9%)
├── 2억~200억: 20% (22%)    ← 대부분 이 구간
├── 200억~3,000억: 22% (24.2%)
└── 3,000억 초과: 25% (27.5%)

→ 순이익 5억 기준: 개인 42~45% vs 법인 20% = 약 20%p 차이
→ 절세 효과: 연간 1억 원 이상
```

### 5-2. 성실신고확인 대상 회피

```
개인사업자 도소매업 매출 15억+ → 성실신고확인 대상자 (288억이면 당연 해당)
법인 전환 시 → 성실신고확인 대상에서 제외
→ 세무조사 선정 위험↓ + 성실신고확인 비용 절감
```

### 5-3. 대표이사 급여 최적화

| 전략 | 연봉 | 소득세 한계세율 | 설명 |
|------|------|--------------|------|
| 보수적 | ~7,000만 | 16.5% | 보험료 최소화 우선 |
| **균형적 (추천)** | **~1.1억** | **26.4%** | **법인세(22%)와 균형** |
| 적극적 | ~1.5억 | 38.5% | 법인 이익 줄이기 우선 |

### 5-4. 배당 전략 (2026년 신설!)

```
2026년 배당소득 분리과세 신설:
├── 2,000만 원 이하: 15.4%
├── 2,000만~3억: 22%
├── 3억~50억: 27.5%
└── 50억 초과: 33%

종전: 2,000만 초과 배당 → 종합과세 최고 49.5%
변경: 분리과세 선택 가능 → 최고 33%

최적 전략 (급여 + 배당 혼합):
├── 급여: 연 1~1.1억 (소득세 26.4% 이내, 법인 비용처리)
├── 배당: 연 2,000만 원 (15.4% 분리과세)
└── 나머지: 법인 내 유보 (법인세 20%)
```

### 5-5. 임원 퇴직금 활용

```
법인 대표이사 퇴직금 한도:
= 퇴직 전 1년간 총급여 x 1/10 x 근속연수

예시: 연봉 1.2억, 근속 20년
= 1.2억 x 1/10 x 20 = 2.4억 원

퇴직소득세: 연분연승 분리과세 → 실효세율 매우 낮음
IRP로 수령 후 연금 인출 시 → 퇴직소득세 추가 30~40% 감면
```
