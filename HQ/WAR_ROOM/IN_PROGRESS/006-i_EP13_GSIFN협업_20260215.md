## 2. GSIFN 사업 모델 이해

### 2-1. GSIFN 추정 사업 영역

```
GSIFN (Global Strategic Intelligence & Financial Network 추정)
├── 정보 중개 / 인텔리전스
├── 금융 네트워크 / 컨설팅
├── B2B 정보 서비스
└── EP-09에서 사장님이 언급: "GSIFN의 정보 중개 모델과 유사한 구조"
    → Echemi RFQ 마켓 모델과 비교하며 언급
```

### 2-2. 사장님 사업(은파우더 무역)과의 접점

```
사장님 강점:
├── 은파우더/귀금속 실물 무역 경험
├── 일본/글로벌 공급망 네트워크
├── 수입/수출 실무 역량
└── 288억 규모 거래 실적

GSIFN 추정 강점:
├── 정보 수집/분석 역량
├── 금융/비즈니스 네트워크
├── 전략적 인텔리전스
└── 글로벌 커넥션
```

## 3. 협업 가능성 시나리오

### 시나리오 A: 정보 공유 + 시장 인텔리전스

```
GSIFN의 정보 분석 역량
    +
사장님의 무역 실물 데이터
    =
시장 동향 리포트 / 바이어 인텔리전스 서비스

예시:
- GSIFN이 글로벌 은 시장 동향 분석
- 사장님이 실거래 가격/물량 데이터 제공
- 합작 리포트 → 업계 판매 또는 자체 활용
```

### 시나리오 B: 바이어/셀러 매칭 서비스

```
GSIFN의 네트워크 + 정보 역량
    +
사장님의 실물 무역 경험 + Echemi/일본 연결
    =
B2B 매칭 플랫폼 또는 중개 서비스

예시:
- GSIFN이 글로벌 바이어 발굴
- 사장님이 공급 능력 + 품질 보증
- 공동 중개 수수료 모델
```

### 시나리오 C: 정보 중개 + 무역 중개 결합

```
GSIFN의 정보 중개 모델
    +
사장님의 EP-09(Echemi) + EP-08(일본) 네트워크
    =
"정보 + 실물" 원스톱 서비스

예시:
- "이 바이어가 은파우더 찾고 있습니다" (정보)
- → "우리가 납품할 수 있습니다" (실물)
- 정보 제공비 + 무역 마진 이중 수익
```

### 시나리오 D: 독일 장비(EP-10) + 정보 서비스

```
GSIFN의 글로벌 네트워크
    +
사장님의 EP-10(독일 장비 사이트) 발견
    =
산업 장비 정보 서비스 / 중개

예시:
- 독일 장비 정보를 한국어로 정리 (사장님)
- GSIFN이 바이어 네트워크에 유통
- 장비 중개 수수료
```

## 4. 주의사항

### 4-1. 법적/세무적 고려

```
⚠️ 부부간 사업 협업 시 주의점:
├── 특수관계자 거래 → 세무 리스크
│   └── 시가와 다른 가격으로 거래 시 부당행위계산 부인
├── 자금 흐름 → 증여세 이슈
│   └── 부부간 6억 원까지 증여세 면제 (10년 기준)
├── 소득 분산 → 합리적 근거 필요
│   └── 실질적 역무 제공 대가여야 함
└── 세무사 상담 필수

✅ 안전한 방법:
├── 정식 업무위탁계약 체결
├── 시장가 기준 대가 지급
├── 업무 내역 + 성과 기록 유지
└── 각각의 법인에서 처리 (부부 개인 간 거래 X)
```

### 4-2. 이해관계 충돌 방지

```
사장님 사업 → 은파우더 실물 무역
GSIFN → 정보/금융 서비스

→ 업역이 다르므로 이해관계 충돌 낮음
→ 오히려 보완적 관계 = 시너지 가능성 높음
→ 단, 고객 정보/영업 비밀 공유 시 별도 NDA 필요
```

## 6. 권고

### 지금 할 것 (호기심 단계)

```
1. GSIFN의 정확한 사업 영역/강점을 사장님이 정리
2. 사장님 사업에서 GSIFN이 도와줄 수 있는 구체적 업무 1~2개 특정
3. 작은 실험: "Echemi RFQ에 같이 대응해보기" 등
4. 시너지 확인 후 → 세무사와 공식 협업 구조 설계
```

### 지금 안 할 것

```
❌ 급하게 합작법인 설립
❌ 대규모 자금 거래
❌ 고객 DB 무분별 공유
→ 호기심 단계에서는 탐색만 하고, 확신이 생기면 구조화
```

## 7. 평가

**전략적 타당성: ★★★☆☆ (3/5) — 탐색 단계**

- **가능성:** 정보(GSIFN) + 실물(사장님) = 보완적 시너지 존재
- **리스크:** 특수관계자 거래 세무 리스크, 사업-가정 분리 필요
- **현재 단계:** 호기심 → 작은 실험 → 확인 후 구조화
- **우선순위:** EP-08/09/11 > EP-07(법인) > EP-13(GSIFN) — 핵심 사업 안정화 먼저
