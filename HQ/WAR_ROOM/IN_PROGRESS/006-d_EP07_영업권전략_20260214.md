## 1. 사장님 브리핑

- 실버 파우더 사업으로 법인 세우게 되면 **영업권(Goodwill)도 최대한 받을 수 있게** 하고 싶다
- 단순 법인 전환이 아니라, **개인사업 시절 쌓은 가치를 영업권으로 인정받아 절세**하겠다는 전략
- EP-07(법인 vs 개인 시뮬레이션)의 후속 — 법인 전환 결정 시 영업권 극대화가 핵심

## 2. 영업권(Goodwill) 기본 구조

### 영업권이란?

```
영업권 = 사업 전체 가치 - 순자산(자산 - 부채)

→ "장부에 안 잡히지만 돈 버는 힘"
→ 거래처 네트워크, 브랜드 신뢰, 기술 노하우, 고객 관계 등
```

### 법인 전환 시 영업권 활용 구조

```
[개인사업자 사장님]
     │
     │  포괄양수도 (사업 통째로 이전)
     │  → 영업권 포함 양도
     ↓
[신설 법인]
     │
     │  영업권을 무형자산으로 계상
     │  → 5년간 감가상각 → 매년 비용 처리 → 법인세 절감
     ↓
[사장님 개인]
     │
     │  영업권 양도대가 수령
     │  → 기타소득 (60% 필요경비 인정)
     │  → 실효세율 약 8.8~17.6%
     ↓
[결과: 법인은 절세, 개인은 저율과세]
```

## 3. 핵심 세무 구조 분석

### 3-1. 영업권 평가 방법: 초과수익환원법

```
영업권 = Σ (초과이익 × 현가계수)  [3~5년간]

초과이익 = 실제 순이익 - (순자산 × 정상수익률 10%)
현가계수 = 1/(1+할인율)^n  (할인율: 통상 10%)
```

**사장님 사업 적용 시뮬레이션:**

| 항목 | 금액 |
|------|------|
| 연매출 | 288억 원 |
| 추정 순이익 (보수적 3%) | 약 8.6억 원 |
| 순자산 (추정) | 약 10억 원 |
| 정상수익 (순자산×10%) | 1억 원 |
| 초과이익 | 약 7.6억 원 |
| **영업권 (5년 환산)** | **약 28.8~34.5억 원** |

> ※ 실제 금액은 감정평가법인이 결정. 위는 개념 시뮬레이션.

### 3-2. 사장님(양도자) 세금

```
영업권 양도 = 기타소득
├── 필요경비: 60% 자동 인정 (증빙 불요)
├── 과세표준: 40%만 과세
├── 세율: 종합소득세 누진세율 (6~45%)
└── 분리과세 선택 가능 (22% 단일세율, 금액 크면 유리)

예시) 영업권 30억 원 수령 시:
├── 필요경비 60%: 18억 원 공제
├── 과세표준: 12억 원
├── 분리과세 22% 선택: 약 2.64억 원
└── 실효세율: 약 8.8%
```

### 3-3. 법인(양수자) 절세 효과

```
영업권 30억 원 무형자산 계상:
├── 5년 균등 감가상각: 연 6억 원 비용 처리
├── 법인세 절감 (세율 20% 가정): 연 1.2억 원
├── 5년간 총 절감: 약 6억 원
└── 추가: 감가상각비는 현금 유출 없는 비용 → 현금흐름 개선
```

### 3-4. 포괄양수도 방식

```
포괄양수도 = 사업 통째로 이전
├── 장점: 부가가치세 면제 (부가가치세법 제10조 9항)
├── 장점: 거래처/계약 자동 승계
├── 장점: 사업 연속성 유지
├── 조건: 사업용 자산 전부 포함 + 사업 계속 영위
└── 주의: 부분 양도는 포괄양수도 불인정 → VAT 과세
```

## 4. 영업권 극대화 전략 (7단계)

### 전략 1: 법인 전환 전 매출/수익 극대화

```
법인 전환 최소 2~3년 전부터 준비:
├── 매출 성장 트렌드 유지 (연 10%+ 성장 기록)
├── 순이익률 개선 (원가 절감, 마진 개선)
├── 신규 거래처 확보 → 영업권 근거 강화
└── EP-08(일본) + EP-09(Echemi) = 거래처 확대 근거
```

### 전략 2: 무형자산 근거 문서화

```
영업권 평가 시 감정평가법인이 보는 것:
├── 거래처 리스트 + 거래 기간 + 거래 금액
├── 독점/준독점 공급 계약
├── 기술 노하우 (은파우더 가공/품질관리)
├── 브랜드 인지도 (업계 내 평판)
├── 인적 네트워크 (일본/글로벌 소싱 라인)
└── 반복 매출 비율 (단골 거래처 비중)

→ 지금부터 정리 시작 필수!
```

### 전략 3: 감정평가법인 선정

```
감정평가법인 선정 기준:
├── 영업권 평가 경험 多 (무역/도소매 업종)
├── 국세청 소명 경험 있는 곳
├── 비용: 300~500만 원 (영업권 규모에 따라)
└── 추천: 최소 2곳에서 평가 받아 비교
```

### 전략 4: 적정 영업권 금액 설정

```
영업권 너무 높으면:
├── 국세청 부당행위계산부인 리스크
├── 세무조사 타겟
└── 감정평가 논리 부족 시 부인 가능

영업권 너무 낮으면:
├── 절세 효과 미미
└── 사장님 양도대가 적음

→ "감정평가법인 산출 금액의 80~100%" 범위가 안전
→ 288억 매출 기준 영업권 20~35억 범위 예상
```

### 전략 5: 양도 타이밍

```
최적 타이밍:
├── 직전 2~3년 실적이 가장 좋을 때
├── 업계 호황기 (귀금속/촉매 수요 상승기)
├── 신규 거래처 확보 직후 (일본 라인 구축 후)
└── 세법 개정 전 (필요경비 60% 혜택 유지 확인)

→ EP-08 일본 소싱 + EP-09 Echemi 안착 후 = 최적 타이밍
```

### 전략 6: 법인 자본금 설계

```
법인 자본금 구성:
├── 현금 출자: 최소한으로 (자본금 100만~1,000만 원)
├── 영업권 양수 대금: 분할 지급 가능 (3~5년)
│   → 법인 현금 부담 분산
│   → 사장님 소득 분산 (연도별 과세)
└── 주의: 가지급금 발생 시 인정이자 과세 리스크
```

### 전략 7: 전문가 팀 구성

```
필수 전문가:
├── 세무사: 영업권 양도소득 신고 + 법인 설립 세무
├── 감정평가법인: 영업권 가치 산정
├── 법무사: 법인 설립 등기
└── (선택) 회계사: 대규모 영업권 시 세무조사 대비

→ 세무사 + 감정평가법인은 반드시 사전 미팅
→ "영업권 극대화하되 국세청 소명 가능한 선"에서 합의
```

## 5. 국세청 리스크 관리

### 국세청이 주목하는 케이스

```
⚠️ 주의 신호:
├── 영업권이 순자산의 5배 이상 → 소명 요구 가능성 높음
├── 감정평가 없이 당사자 간 합의만으로 금액 결정
├── 법인 전환 직후 영업 실적 급감 → "실체 없는 영업권" 의심
├── 특수관계자 간 거래 (개인 → 본인 법인) → 부당행위계산부인 리스크
└── 분할 지급 약정 미이행 → 가지급금 과세
```

### 대응 전략

```
✅ 안전 장치:
├── 감정평가법인 공식 보고서 확보 (필수)
├── 거래처 리스트/계약서 등 영업권 근거 서류 구비
├── 포괄양수도 계약서 공증
├── 법인 전환 후에도 매출 유지/성장 실적 확보
├── 영업권 분할 지급 시 실제 이체 기록 남기기
└── 세무사와 사전 유권해석 신청 검토
```

## 8. 평가

**전략적 타당성: ★★★★★ (5/5)**

- 288억 매출 → 영업권 20~35억 범위 예상
- 개인: 실효세율 8.8% (분리과세)
- 법인: 5년간 약 6억 원 법인세 절감
- EP-08(일본) + EP-09(Echemi) = 영업권 근거 강화 수단
- 리스크: 감정평가 없으면 국세청 부인 가능 → 감정평가 필수

> **최실장:** "EP-07(법인 전환) + EP-07 보강(영업권) = 법인 설립의 완전한 전략.
> 핵심은 '지금부터 근거 자료 쌓기'. 법인 세울 때가 아니라 지금 시작해야 합니다."
