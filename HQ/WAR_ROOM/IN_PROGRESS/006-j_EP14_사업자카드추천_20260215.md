## 4. 법인카드 vs 개인사업자 카드 비교

| 항목 | 법인카드 | 개인사업자 카드 |
|------|----------|----------------|
| **발급 명의** | 법인 | 대표자 개인 |
| **홈택스 등록** | 자동 등록 | 별도로 사업용 카드 등록 필요 |
| **적격 증빙** | 자동 인정 | 등록 시 인정 |
| **사적 사용** | **엄격히 금지** (상여처분/가지급금) | 혼재 가능하나 구분 필수 |
| **비용처리** | 법인 경비 자동 | 사업관련만 비용처리 |
| **한도** | 법인 신용도 기준 (높음) | 개인 신용도 기준 |
| **포인트** | 법인 귀속 (개인 사용 제한) | 개인 귀속 (자유 사용) |

```
★ 결론:
├── 현재(개인사업자): 개인 카드를 홈택스에 사업용 등록하여 사용
├── 법인 전환 후: 법인카드 발급 → 사업비 전용 / 개인카드는 개인용으로 분리
├── 핵심: 사업용 카드와 개인용 카드를 반드시 분리!
└── 카드사 PB/VVIP 담당에게 맞춤 한도 협의 (288억 매출이면 VVIP 대상)
```

## 5. 해외결제 수수료 비교 (무역업 핵심)

### 5-1. 카드사별 해외이용 수수료율

| 수수료율 | 카드사 |
|----------|--------|
| **0.18%** (최저) | 기업, 신한, 현대 |
| 0.20% | 광주, 롯데, 삼성, 전북, 하나 |
| 0.25% | 국민, 농협 |
| 0.30% | 수협, 우리, 제주 |

> 일반적으로 해외 결제 시 총 **약 2.5% 수수료** 발생
> (국제브랜드 + 환전 + 카드사 수수료)

### 5-2. 해외수수료 실질 0% 카드

```
1. 삼성 iD GLOBAL — 총 수수료 1.2%, 해외 2% 할인 → 실질 마이너스(이득!)
2. 신한 Globus — 수수료 전액 할인 + 1.7% 적립 → 실질 0%
3. BC 바로 MACAO — 해외 전 가맹점 2% 할인(무제한) → 실질 마이너스
```

## 6. 세금 납부 카드 전략

### 6-1. 세금 카드 납부 수수료

| 구분 | 수수료 |
|------|--------|
| 국세 (부가세, 소득세, 법인세) 신용카드 | **0.8%** |
| 국세 체크카드 | **0.5%** |
| 지방세 (재산세 등) | **무료** |

### 6-2. 무이자 할부

| 카드사 | 할부 |
|--------|------|
| **BC카드** | 국세 3개월 무이자 (사실상 유일) |
| 신한 | 슬림할부 (부분 무이자) |

### 6-3. 전략

```
지방세: 수수료 무료 → 적립률 높은 카드로 납부 (이득)
국세: 0.8% 수수료 → 대부분 현금(계좌이체) 납부가 유리
      단, 현금 유동성 필요 시 → BC카드 3개월 무이자 활용
```

> **트럼프 이사:** "288억 매출이면 부가세만 해도 엄청납니다.
> 0.8% 수수료의 절대 금액이 크니까 국세는 계좌이체가 맞고,
> 지방세만 카드로 납부해서 포인트 챙기세요."
