## 2. Echemi 플랫폼 상세 분석

### 2-1. 가입 및 등록 절차

```
Step 1: 회원가입
├── echemi.com 접속 → "Sign Up" 또는 "Register"
├── 이메일 / 비밀번호 / 회사명 / 국가 입력
├── 이메일 인증 → 기본 계정 활성화
└── 소요 시간: 5분

Step 2: 공급사(Seller) 프로필 설정
├── Company Type 선택:
│   ├── Manufacturer (제조사)
│   ├── Trading Company (무역업체) ← 사장님 해당
│   ├── Distributor (유통업체)
│   └── Agent (중개인)
├── Business License 업로드 (사업자등록증)
├── 회사 소개 / 로고 / 대표 사진
├── 취급 제품 카테고리 선택
└── 인증 완료: 1~3 영업일

Step 3: 제품 등록
├── 제품명 / CAS Number / 규격 / 순도
├── 가격 (FOB/CIF/협의가능)
├── 최소주문량(MOQ) / 포장 단위
├── 제품 사진 / SDS(안전보건자료) / COA(성적서)
├── 납기 / 출하지
└── 1개 제품 등록 소요: 15~30분
```

### 2-2. 은파우더 관련 HS Code & 카테고리

```
은파우더 HS Code: 7106.10
├── 7106: Silver (including silver plated with gold or platinum)
├── 7106.10: Powder (은 분말)
└── 7106.91: Unwrought (반제품)

Echemi 카테고리 매핑:
├── Inorganic Chemicals → Silver Compounds
├── Metal & Metal Products → Precious Metals
├── Catalysts → Precious Metal Catalysts
└── 관련 키워드: Silver Powder, Silver Nanoparticles, Silver Catalyst
```

### 2-3. RFQ(견적요청) 시스템 상세

```
RFQ 작동 원리:
├── 바이어가 RFQ 등록 시 포함 정보:
│   ├── 제품명 / CAS Number
│   ├── 필요 수량 / 순도 / 규격
│   ├── 목적지 국가
│   ├── 납기 희망일
│   ├── 예산 범위 (선택)
│   └── 추가 요구사항 (인증/포장 등)
│
├── 공급사(Seller) 대응 방법:
│   ├── RFQ 리스트 브라우징 (카테고리/키워드 필터)
│   ├── 관심 RFQ에 견적(Quote) 제출
│   │   ├── 단가 / MOQ / 납기 / 결제조건
│   │   ├── 제품 상세 스펙
│   │   └── 회사 소개 (차별화 포인트)
│   ├── 바이어가 비교 검토 → 선택 → 1:1 협상
│   └── 거래 성사 시 Echemi 플랫폼 외부에서도 거래 가능
│
├── RFQ 유형:
│   ├── Open RFQ: 모든 공급사에 공개
│   ├── Targeted RFQ: 특정 공급사에만 노출
│   └── Urgent RFQ: 긴급 수요 (보통 프리미엄 가격)
│
└── 핵심 팁:
    ├── 견적 응답 속도가 빠를수록 선정 확률 UP
    ├── CAS Number 매칭이 정확할수록 노출 UP
    ├── 제품 인증서(COA/SDS) 첨부 시 신뢰도 UP
    └── 최초 거래는 소량(샘플)부터 → 신뢰 구축
```

## 3. 은파우더 시장 기회 분석

### 3-1. 글로벌 은파우더 수요 동향

```
주요 수요처:
├── 태양광 패널 (Silver Paste) — 최대 수요처
│   └── 태양광 1GW 설치 시 은 약 20톤 소요
├── 전자부품 (도전성 페이스트/접착제)
├── 촉매 (화학반응 촉매, 에틸렌옥사이드 등)
├── 의료/항균 (나노실버)
├── 3D 프린팅 (금속 분말)
└── 배터리 (Silver-Zinc 배터리)

시장 규모:
├── 글로벌 은파우더 시장: 약 $2.5B (2025)
├── 연간 성장률: 5~7%
└── 주요 성장 드라이버: 태양광 + 전자부품
```

### 3-2. 중국 은 수출 규제 (2026년 1월~)

```
⚠️ 핵심 기회:

중국 정부 2026년 1월부터:
├── 안티몬, 갈륨, 게르마늄 수출 규제 강화
├── 희토류 수출 허가제 확대
├── 은(Silver) 관련 수출 모니터링 강화 조짐
└── 배경: 미중 무역 분쟁 + 전략 광물 통제

→ 중국 공급 불안정 → 대체 공급처 수요 급증
→ 일본/한국 소싱 라인 보유한 사장님에게 기회
→ Echemi RFQ에서 "중국 외 소싱처" 찾는 바이어 증가 예상
```

### 3-3. 은파우더 Echemi 등록 전략

```
등록할 제품 라인:

1. Silver Powder (은 분말)
   ├── CAS: 7440-22-4
   ├── 순도: 99.9% / 99.99% / 99.999%
   ├── 입자 크기: 1~10μm / 10~50μm / 50~100μm
   ├── 용도: 전자/태양광/촉매
   └── MOQ: 1kg~

2. Silver Flake (은 플레이크)
   ├── 도전성 페이스트용
   └── 전자부품 시장 타겟

3. Silver Catalyst (은 촉매)
   ├── 에틸렌옥사이드 제조용
   └── 화학 플랜트 타겟

4. Silver Nanoparticles (은 나노입자)
   ├── 항균/의료용
   └── 고부가가치

→ 하나의 원료를 4가지 제품으로 세분화 → 노출 극대화
```

## 5. 멀티 플랫폼 전략

### 5-1. B2B 플랫폼 우선순위

```
Tier 1 (즉시 등록):
├── Echemi — 화학 특화, RFQ 활발
├── TradeKorea — KITA 운영, 한국 기업 무료
└── BuyKorea — KOTRA 운영, 정부 지원

Tier 2 (1개월 내):
├── EC21 — 한국 B2B 최대, 무료 기본
├── Alibaba — 최대 트래픽, 유료 필요
└── Made-in-Korea — 한국 제조 특화

Tier 3 (분석 후):
├── IndiaMART — 인도 시장 (은 수요 大)
├── GlobalSources — 아시아 전문
└── Tridge — 농산물+화학 B2B
```

### 5-2. 정부 지원 프로그램

```
KOTRA 지원:
├── BuyKorea 입점 무료
├── 해외 바이어 매칭 프로그램
├── 화상 상담회 (연 수회)
└── 수출 바우처 (마케팅 비용 지원)

KITA 지원:
├── TradeKorea 기본 무료
├── 해외 전시회 참가 지원
├── 무역 인큐베이팅
└── FTA 활용 컨설팅

중소기업진흥공단:
├── 수출 초보 기업 지원
├── 해외 마케팅 비용 보조
└── 수출 보험 가입 지원
```
