## 1. 사장님 지시 (원문)

> "사장님이 숫자를 보며 고민하시게 하지 마라.
> **'원인과 해결책'**이 포함된 보고서만 사장님께 올려라."

## 2. 기존 보고 vs 변경 보고

### 2-1. 기존 (숫자 나열형) ❌

```
[주간 보고 — 2월 3주차]

- 매출: 2,450만원
- 매입: 1,800만원
- 지출: 850만원
- 리베이트: 525만원
- 잔액: 3,200만원

이상입니다.
```

**문제:** 사장님이 직접 "왜 지출이 늘었지?" 분석해야 함

### 2-2. 변경 (원인+해결책형) ✅

```
[주간 보고 — 2월 3주차]

■ 핵심 요약
  매출 2,450만 / 매입 1,800만 / 순이익 650만
  전주 대비 지출 +15% (850만 → 978만)

■ 주의 항목
  ⚠️ 광고비 과다: 200만 → 350만 (+75%)
     원인: B채널 광고 단가 인상 (CPC 800원→1,400원)
     해결: B채널 ROI 0.8배로 적자 → 즉시 중단 권고
     절감 효과: 월 150만 절약

  ⚠️ 재고 부족 임박: YY원료 (현재 20EA, 최소 30EA)
     원인: 2월 납품 3건으로 소진 가속
     해결: C상사에 50EA 긴급 발주 (예상 원가 225만)

■ 긍정 항목
  ✅ A제약 거래처 신규 확보 → 월 300만 추가 매출 기대
  ✅ 리베이트 정산 순조 (홍길동 완료, 김철수 2/20 예정)

■ 결재 필요 사항
  □ B채널 광고 중단 (절감 150만/월)
  □ YY원료 긴급 발주 50EA (225만)

이상입니다.
```

## 4. 클로드에게 던질 프롬프트 모음 ★핵심★

### 4-1. 주간 보고서 생성

```
아래 형식으로 이번 주 보고서를 작성해줘:

■ 핵심 요약: 매출/매입/순이익, 전주 대비 변동률
■ 주의 항목: 이상 징후 + 원인 + 해결책 (각 항목마다)
■ 긍정 항목: 좋은 변화
■ 결재 필요 사항: 사장님이 결정해야 할 것

숫자만 나열하지 말고, 반드시 "왜"와 "어떻게 해결할지"를 포함해줘.
```

### 4-2. 현금흐름 분석

```
현금흐름 시트를 분석해서 알려줘:
1. 이번 달 총 입금액 / 총 출금액 / 잔액 변화
2. 가장 큰 출금 항목 TOP 3와 그 비율
3. 전월 대비 비정상적으로 증가한 항목이 있으면 원인 추정
4. 다음 달 예상 잔액 (현재 추세 기준)
```

### 4-3. 리베이트 정산

```
리베이트정산 시트를 보고 알려줘:
1. 이번 달 정산 대상자 리스트
2. 각 대상자별 정산액
3. 총 정산액
4. 미지급 상태인 건이 있으면 표시
5. 지급 예정일 기준 일정 정리
```

### 4-4. 재고 경보

```
재고현황 시트에서:
1. 현재수량이 최소수량보다 적은 품목 전부 뽑아줘
2. 각 품목별 부족분(최소수량 - 현재수량) 계산
3. 평균원가 기준 긴급 발주 시 예상 비용 계산
4. 가장 시급한 순서대로 정렬
```

### 4-5. 매입 원가 분석

```
매입이력 시트에서:
1. 최근 3개월간 품목별 평균 매입 단가 추이
2. 단가가 10% 이상 오른 품목 표시
3. 공급사별 거래 빈도와 총액 순위
4. 운임 비중이 높은 거래 식별
```

### 4-6. 수익성 분석 (매입 + 납품 교차)

```
매입이력과 납품이력 시트를 교차 분석해서:
1. 품목별 매입 원가 vs 판매 단가 → 마진율 계산
2. 마진율이 가장 높은/낮은 품목 TOP 5
3. 마진이 마이너스인 품목이 있으면 즉시 알려줘
4. 거래처별 총 매출액과 수익 기여도 순위
```

### 4-7. 비정상 탐지 (범용)

```
전체 시트를 훑어보고:
1. 평소와 다른 패턴이 있는지 찾아줘
2. 갑자기 늘거나 줄어든 수치
3. 빈 셀이나 이상한 값
4. 내가 놓칠 수 있는 위험 신호
```

## 7. 주의사항

```
⚠️ 클로드 답변을 그대로 복사하지 마라
   → 클로드가 모르는 맥락(사장님 의중, 거래처 관계 등) 반영 필수
   → 최실장이 반드시 검토 후 보고

⚠️ 수치는 반드시 검증
   → 클로드가 계산한 숫자를 엑셀 원본과 대조
   → 특히 합계, 비율, 전주 대비 변동률

⚠️ 민감 정보 주의
   → 리베이트 대상자 실명 → 보고서에는 포함하되 외부 유출 금지
   → 클로드 채팅은 세션 종료 시 삭제됨 (30일 후 서버에서도 삭제)
```
