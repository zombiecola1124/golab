## 0. 사장님이 알려주신 것 (불변 조건)

```
1. 리베이트 구조 (건별):
   ├── 수익금의 30% → 세금 준비금
   ├── 나머지 70%의 60% → 현금 리베이트 (한 군데)
   └── 나머지 70%의 40% → 내 돈

2. 와이프: 월급이 아니라 생활비 300만/월

3. 차량: 펠리세이드 9인승 장기렌트 4년 (2026년 1월, 개인사업자 명의)

4. 법인 전환 시 사장님 월급: 최소 300만

5. 고정지출: 현재 개인사업자 명의로 걸려 있음
```

## 1. 현재 돈 흐름 (개인사업자)

### 연간 기준 (이익 1.5억)

```
매출: 288억
이익: 1.5억 (마진 0.5%)

건별 분배 룰 적용:
┌─────────────────────────────────────────────────┐
│  이익 1.5억                                      │
│                                                  │
│  ① 세금 준비 30%          = 4,500만              │
│                                                  │
│  ② 잔금 70%               = 1.05억               │
│     ├── 60% 현금 리베이트  = 6,300만 (월 525만)   │
│     └── 40% 내 돈          = 4,200만 (월 350만)   │
│                                                  │
│  내 돈 350만/월 에서:                             │
│     ├── 와이프 생활비       = 300만               │
│     └── 사장님 생활비       = 50만                │
└─────────────────────────────────────────────────┘

실제 세금 (종소세+지방세): ~4,077만
세금 준비금과 차이: 4,500 - 4,077 = +423만 여유

4대보험 (지역가입자): ~1,948만/년 (월 ~162만)
→ 이건 어디서 나옴? 내 돈 4,200만에서 빠짐

실제 사장님 수중:
  4,200만(내 돈)
- 3,600만(와이프 생활비)
- 1,948만(4대보험)
─────────────
= ▲1,348만 부족

세금 준비금 여유분 423만 돌려도: ▲925만 부족
```

### 문제 발견

```
★ 현재 구조에서 사장님 생활비가 마이너스임 ★

이익 1.5억으로는:
  세금 4,077만 + 보험 1,948만 + 리베이트 6,300만 + 와이프 3,600만
  = 15,925만 > 15,000만 (이익)

→ 연간 925만 원 부족 (월 ~77만 적자)
→ 사장님이 실제로 쪼들리고 있었던 이유가 여기 있음
```

## 2. 펠리세이드 장기렌트

```
차량: 펠리세이드 9인승
계약: 2026년 1월, 개인사업자 명의, 4년
월 렌트비: 약 90~120만 (추정, 옵션에 따라 다름)
연간: 약 1,080~1,440만

비용처리 (개인사업자):
├── 9인승 = 업무용 차량으로 인정 (운행일지 불필요!)
├── ★ 9인승 이상은 연 1,500만 한도 적용 안 됨 ★
├── 렌트비 전액 비용처리 가능
└── 이게 사장님이 말한 "개인으로 때려넣으면 세금 줄어들잖아"

세금 절감 효과 (세율 35% 구간 기준):
├── 연 렌트비 1,200만 x 35% = 약 420만 원 세금 절감
└── 소득세 + 지방세 포함하면 약 460만 원 절감

법인 전환 시 이 차량:
├── 개인사업자 명의 계약 → 법인 비용처리 불가
├── 선택지 ①: 렌탈사에 명의변경 요청 (가능 여부 확인 필요)
├── 선택지 ②: 4년 만기까지 개인 유지, 이후 법인 재계약
├── 선택지 ③: 법인이 사장님에게 차량 사용료 지급 (전대 구조)
└── ★ 추천: 선택지 ② (만기까지 개인 유지가 가장 깔끔)
```

## 5. 와이프 생활비 → 급여 전환 시 추가 절세

```

├── 와이프 실수령: 월 ~266만 (현재 300만보다 줄어듦)

├── ★ 조정: 월급 350만으로 세팅하면 실수령 ~300만 맞출 수 있음
```
