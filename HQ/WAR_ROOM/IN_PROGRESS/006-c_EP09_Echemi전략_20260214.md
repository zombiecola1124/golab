## 1. 사장님 브리핑

- **Echemi** (echemi.com) 발견 — 글로벌 화학원료 B2B 플랫폼
- 전세계 사람들이 "이거 필요합니다" 올리는 구조
- 한국 업체들도 올리고 있음 → "한국에 없으니까 올렸겠지?"
- **빠르게 캐치해서 판매/홍보** 가능성

**추가 정보:**
- 와이프 회사 GSIFN이 IR서비스 하는 곳인데, Echemi의 정보 중개 구조와 유사한 점 발견
- Echemi는 수출입 데이터도 활용하는 것 같음

## 2. Echemi 플랫폼 분석 (조사 결과)

### 개요

| 항목 | 내용 |
|------|------|
| 설립 | 2015년, 홍콩 본사 |
| 규모 | 바이어 12만+ / 공급사 2.3만+ / 200개국 |
| 제품 | 100만+ 제품, 200+ 카테고리 |
| 월 문의 | 11,500건+ |
| 한국법인 | ECHEMI KOREA (2023년 11월 설립) |
| ICIS | 글로벌 화학 유통사 Top 50 선정 |

### 핵심 기능 3가지

```
① RFQ 마켓 (수요 신호 탐지기)
   → 전세계 바이어가 "이거 필요합니다" 올림
   → 공급자가 보고 견적 제출
   → 사장님이 보신 핵심 기능

② 무역 데이터 (경쟁사 모니터링)
   → 218개국, 100억건+ 실제 통관 데이터
   → 경쟁사 수출 단가/물량/거래처 확인 가능
   → 91개국 상세 세관 데이터

③ 시세 정보 (시장 타이밍)
   → 200개+ 품목 실시간 가격 추이
   → 주간/월간 리포트
   → "지금이 사는 타이밍인가" 판단 근거
```

### RFQ 실제 사례

- 인듐 잉곳 (7N 순도) 구매 요청
- 이산화티타늄 — 우간다 소싱 회사, 장기 공급사 찾음
- 의약품 원료 — CAS번호 기반 5kg, 납기/가격/순도 요청
- 나프텐산마그네슘 — 호주/프랑스/페루/방글라데시/터키 다수
- 옥시염화비스무트 — 프리미엄 소싱

### 이미 활동 중인 한국 업체

- **KCC** (서울, 1962년 설립, 100개국+ 영업)
- **Chems Korea** (석유화학 유통, 국내 100개+ 거래처)
- **Mechem Korea** (니켈, 페로얼로이, 코발트 — 배터리 산업)
- Korea PTG, ENC GLOBAL 등

## 3. 전 팀 회의 결과

### 사장님 사업과의 연결점

```
사장님 발견 구조:

전세계 바이어 → "이거 필요해요" RFQ 올림
                     ↓
사장님이 캐치 → "내가 조달할 수 있어"
                     ↓
EP-08 일본 소싱 → 납품 → 마진
```

### 활용 방법 2가지

```
방법 A: 바이어로 (소싱)
→ 일본/중국 공급사 발굴 → 더 싼 매입처 확보

방법 B: 셀러로 (판매)
→ 사장님 제품 등록 → 글로벌 바이어 노출
→ 은파우더/귀금속 촉매 카테고리 존재
```

### EP-08 + EP-09 시너지

```
EP-08 (공급측): 일본 전시회 → 새 소싱처 발굴
EP-09 (수요측): Echemi RFQ → 누가 뭘 찾는지 탐지

합치면:
"일본에서 소싱 → Echemi에서 바이어 매칭"
"Echemi 수요 캐치 → 일본에서 조달"
→ 공급 + 수요 양쪽 다 잡는 구조
```

### 비용

```
RFQ 열람/대응:     무료
공급사 기본 등록:   무료
VIP 멤버십:        유료 (제품 홍보 강화)
무역 데이터:       유료 구독
→ 초기 비용 0원에서 시작 가능
```
