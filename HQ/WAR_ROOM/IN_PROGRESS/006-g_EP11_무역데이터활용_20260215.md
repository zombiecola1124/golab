## 2. 통관데이터란 무엇인가

### 2-1. 정의

**통관데이터(Customs Data)** = 수출입 화물이 세관을 통과할 때 제출되는 서류/기록의 총체

```
핵심 서류 3가지:
├── 선하증권 (B/L, Bill of Lading)
│   └── 수출자(Shipper), 수입자(Consignee), 화물 내역, 출발/도착항
├── 수출입신고서
│   └── HS코드, 수량, 금액, 원산지
└── 적하목록 (Manifest)
    └── 선박/항공기 적재 화물 총 목록
```

### 2-2. 포함 정보

| 항목 | 설명 |
|------|------|
| HS코드별 수출입 실적 | 품목 분류 코드별 수량, 금액, 중량 |
| 수출자/수입자 정보 | 회사명, 주소 (**국가에 따라 공개 여부 다름**) |
| 원산지 | 화물의 원산지 국가 |
| 거래 단가/총액 | 신고 가격 |
| 운송 경로 | 출발항, 도착항, 경유지 |
| 운송 수단 | 선사, 항공사, 선박명 등 |

### 2-3. 국가별 바이어/셀러 정보 공개 여부 (핵심!)

| 구분 | 미국 | 한국 | 중국 | 일본 | 인도 |
|------|------|------|------|------|------|
| **B/L 데이터 공개** | **공개 (Public)** | 비공개 | 부분 공개 | 비공개 | 부분 공개 |
| **수입자 이름 조회** | 가능 | 불가 | 일부 가능 | 불가 | 일부 가능 |
| **수출자 이름 조회** | 가능 | 불가 | 일부 가능 | 불가 | 일부 가능 |
| **집계 통계** | 공개 | 공개 | 공개 | 공개 | 공개 |

> **미국이 특별한 이유**: 미국 관세국경보호청(CBP)은 수입 B/L 데이터를 대중에게 공개함.
> → ImportGenius, Panjiva 등이 이걸 수집해서 상업적으로 판매하는 것.
>
> **한국이 비공개인 이유**: 관세청이 개별 기업 수출입 신고 데이터를 영업비밀/개인정보 보호 차원에서 비공개.
> → 공개되는 건 HS코드별 집계 통계(총 수량, 총 금액)뿐.

| 2 | **트레이드잇 (TradeIt)** | 해외영업+B/L 빅데이터 (tradeit.co.kr) | 기본 무료 + 프리미엄 | B/L 기반 무역 빅데이터, 바이어 리스트, TaaS. 글로벌 바이어 500개+ |
| 3 | **가이온** | 빅데이터 분석 플랫폼 | 컨설팅+플랫폼 | KITA와 협력, 바이어 발굴용 빅데이터 분석 |

> **사장님 질문 "한국에 통관데이터 장사하는 업체 많아?"에 대한 답:**
>
> **많지는 않습니다.** 한국은 개별 거래 데이터(바이어/셀러 이름)가 비공개이기 때문에,
> 민간 업체가 한국 통관데이터를 직접 판매하기 어렵습니다.
>
> 한국 업체들은 주로:
> 1. **해외 데이터**(미국 B/L 등)를 가공해서 판매하거나 (트레이드잇)
> 2. **물류 서비스**와 데이터를 번들로 제공하거나 (트레드링스)
> 3. **공공 데이터**를 분석/가공해서 컨설팅하는 형태 (가이온)
>
> 글로벌 시장에서는 S&P Global(Panjiva+GTA)이 압도적이고,
> 중소기업 시장은 ImportGenius가 공략 중.

## 4. 은파우더(HS 7106.10) 관련 무역 데이터

### 4-1. 관련 HS Code

| HS Code | 품목 | 설명 |
|---------|------|------|
| **7106.10** | 은 분말 (Silver Powder) | **사장님 핵심 품목** |
| 7106.91 | 은 괴 (미가공) | 은 미가공 형태 |
| 7106.92 | 은 반가공 | 은 반가공 형태 |
| CAS | 7440-22-4 | 은 화학물질 등록번호 |

### 4-2. 글로벌 시장 현황 (2023~2024)

```
은 분말(710610) 주요 수출국 (2024년):
┌─────────────────────────────────────────────┐
│ 1위  일본      $11.1억  (3,531톤)  ← 압도적   │
│ 2위  미국      $4.99억  (547톤)              │
│ 3위  스위스    $5,092만                      │
│ 4위  독일      $4,174만  (45톤)              │
│ 5위  한국      $3,950만  (373톤)  ← 사장님    │
└─────────────────────────────────────────────┘

최대 수입국 — 중국 (2023년):
┌─────────────────────────────────────────────┐
│ 총 수입: $9.97억 (2,726톤)                    │
│ 일본산 89% ($8.89억)                          │
│ 미국산 9% ($8,903만)                          │
│ 한국산 1% ($965만, 107톤) ← 성장 여지 매우 큼   │
└─────────────────────────────────────────────┘

은(HS 7106) 전체 글로벌 수출: $349억 (2024, 전년비 +15.4%)
```

### 4-3. 핵심 인사이트

```
1. 중국 시장 = 가장 큰 기회
   - 중국의 은 수입 중 은 분말(710610)이 84% 차지
   - 2026년 중국 은 수출 규제 → 대체 공급처 수요 급증
   - 한국 점유율 겨우 1% → 확대 여지 막대

2. 일본 독주 구조
   - 일본이 수출의 절대 다수 차지
   - EP-08(일본 전시회)에서 일본 공급망 파악 중 → 시너지

3. 인도 = 차세대 시장
   - 2024년 은 전체 최대 순수입국 ($53억 무역적자)
   - 태양광/전자산업 성장으로 은 수요 폭증
```

## 5. 사장님이 무역데이터로 할 수 있는 것

### 5-1. 무료로 할 수 있는 것 (지금 당장)

### 5-2. 유료 서비스 활용 시 ROI

| 서비스 | 비용 | 얻을 수 있는 것 | ROI 판단 |
|--------|------|----------------|----------|
| **ImportGenius USA Pro** | $199/월 (~26만원) | 미국 수입업체 이름+주소+선적량+공급업체 | 미국 타깃 시 바이어 1건으로 회수 |
| **Tendata iTrader** | $200~500/월 추정 | 65개국 바이어/셀러 상세 거래내역 | 다국가 공략 시 유용 |

### 5-3. 사장님 사업과의 연계 — 4단계 전략

```
1단계: 시장 파악 (무료, 1주일)
├── K-stat에서 한국 은 분말 수출입 현황 파악
├── Trade Map에서 글로벌 수입 상위국 확인
├── WITS에서 중국/미국/인도 수입 추이 확인
└── 현재 내 고객 vs 시장 데이터 비교 → 빠진 시장 발견

2단계: 바이어 발굴 (무료~저비용, 2~4주)
├── 트레이드잇(무료) B/L 기반 바이어 리스트 확보
├── tradeKorea 바이어 매칭
├── KOTRA TriBIG에서 유망시장 확인
└── Echemi RFQ에서 은 분말 수요 모니터링 (EP-09 연계)

3단계: 타깃 집중 (유료, 1~3개월)
├── 미국 공략 시: ImportGenius ($199/월)
├── 다국가 공략 시: Tendata 검토
├── KOTRA 수출바우처 신청 (정부 70~90% 지원)
└── 일본 전시회 데이터 + 무역데이터 교차 분석 (EP-08 연계)

4단계: ERP 연동 (개발팀, 장기)
├── 관세청 공공데이터 API → ERP 대시보드에 시장 데이터 표시
├── K-stat API → 품목별 시장 동향 실시간 표시
├── TRASS My CTI → 수출입실적 자동 연계
└── 은 시세 데이터 → 매입 원가 대비 시장가 비교
```
