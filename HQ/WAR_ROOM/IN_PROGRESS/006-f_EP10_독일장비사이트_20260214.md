## 4. 사업 연결 가능성 분석

### 4-1. 장비 중개 비즈니스 모델

```
모델 A: 독일 장비 → 한국/아시아 수입 중개
├── 독일 중고 장비 (Surplex/Machineseeker 소싱)
├── 한국/동남아 제조업체에 판매
├── 마진: 장비가의 5~15%
├── 장점: 고단가 (장비 1대 수천만~수억 원)
├── 리스크: 물류 복잡, 검수 필요, AS 문제
└── 난이도: ★★★★☆

모델 B: 한국/일본 장비 → 글로벌 판매 (역방향)
├── 한국 장비 (Komachine) + 일본 장비 (EP-08)
├── 글로벌 바이어에 판매
├── Echemi/Alibaba/Surplex에 리스팅
└── 난이도: ★★★☆☆

모델 C: 장비 정보 홍보 + 수수료 모델
├── 독일/일본 장비 정보를 한국어로 정리
├── 블로그/SNS로 홍보
├── 문의 오면 중개 수수료
├── 난이도: ★★☆☆☆
└── → 사장님이 말씀하신 "홍보" 모델과 가장 일치
```

### 4-2. 은파우더 사업과의 연결

```
은파우더 제조/가공에 필요한 장비:
├── 아토마이저 (Atomizer) — 은 분말 제조
├── 볼밀 (Ball Mill) — 분쇄/혼합
├── 분급기 (Classifier) — 입자 크기 분류
├── 진공 건조기 — 분말 건조
├── 체질기 (Sieve) — 입도 분류
└── 분석 장비 (XRF, SEM 등)

→ 독일 장비 사이트에서 중고 은파우더 제조 장비 소싱 가능
→ 자체 제조 능력 확보 시 마진 대폭 향상
→ 또는 이런 장비 필요한 업체에 중개
```
