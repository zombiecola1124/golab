## 1. 사장님 지시 원문 (영상 브리핑 기반)

**비서실 최실장, 사장님 지시 접수:**

사장님께서 'Claude in Excel' 영상을 시청하시고, 해당 기술의 파괴력을 확인하셨습니다.
엑셀이라는 창고 안에 '클로드'라는 초엘리트 직원을 상주시키는 방법입니다.

### 사장님이 주목하신 핵심 기능

```
① 시트 전체를 한 번에 읽기
   → 1번 시트 부품명 + 5번 시트 단가 + 10번 시트 재고를 동시에 파악

② 말귀 알아듣기
   → 함수 대신 "야, 지금 제일 비싼 부품 뭐야? 재고 얼마나 남았어?"
   → 채팅하듯 물어보면 답이 나옴

③ 복합 분석 (Multi-step)
   → "단가 비싼데 불량률까지 높은 부품" 스스로 찾아서
   → "사장님, 이 업체 계약 끊어야 합니다"까지 제안

④ 원가 계산의 마법
   → 계층별 BOM 구조를 파악해서 최종 원가를 5초 만에 계산

⑤ 보고서 자동 작성
   → "재고 부족 품목 TOP 5 + 예상 조달 비용" 즉석 요약
```

### 사장님 3대 지시

| 번호 | 지시 내용 | 담당 |
|------|----------|------|
| 지침 1 | Excel에 클로드 설치, 사장님은 질문만 하면 되게 | 기술본부 최차장 |

## 3. 지침 1 실행 계획: Excel에 클로드 설치

**담당: 기술본부 최차장**

### 3-1. 설치 순서

```
Step 1. Claude 계정 가입 (Pro 플랜 $20/월 결제)
        → claude.ai 에서 가입
        → 사장님 이메일로 Pro 구독

Step 2. Excel에서 추가 기능 설치
        → Excel 열기
        → [홈] → [추가 기능] → "Claude" 검색 → 설치
        → Mac이면: [도구] → [추가 기능]

Step 3. 로그인
        → 오른쪽에 뜨는 사이드바에서 Claude 계정 로그인
        → 끝.

Step 4. 사장님께 보고
        → "사장님, 이제 엑셀 켜고 오른쪽 창에 질문만 하시면 됩니다"

단축키: Ctrl+Alt+C (Windows) / Ctrl+Option+C (Mac)
```

### 3-2. 설치 대상

```
① 사장님 PC (메인)     → 최우선
② 사령부 공용 PC       → 사장님 계정으로 로그인
③ 사장님 노트북 (이동용) → 동일 계정으로 자동 동기화
```

### 3-3. 주의사항

```
⚠️ 엑셀 데스크톱 버전 필요 (웹 엑셀은 미지원)
⚠️ .xlsx 또는 .xlsm 파일만 지원
⚠️ 채팅 기록은 세션 간 유지 안 됨 (매번 새로 시작)
⚠️ 민감 데이터(리베이트 등) → 추후 보안 정책 별도 검토
```

## 4. 지침 2 실행 계획: 장부 표준화

**담당: 재무팀 서차장**

### 4-1. 왜 표준화가 필요한가

```
클로드가 멍청한 대답을 하는 경우:
  ① 열 제목이 "금액1", "금액2" → 뭐가 뭔지 모름
  ② 날짜가 "2/16", "2026.02.16", "16일" 섞여 있음
  ③ 빈 행이 중간에 들어가 있음
  ④ 시트 이름이 "Sheet1", "Sheet2" → 무슨 시트인지 모름

클로드가 똑똑한 대답을 하려면:
  ① 열 제목이 명확: "날짜", "품목명", "수량", "단가(원)", "합계(원)"
  ② 날짜 형식 통일: 2026-02-16
  ③ 빈 행 없이 깔끔한 테이블
  ④ 시트 이름 명확: "재고현황", "매입이력", "리베이트정산"
```

## 5. 지침 3 실행 계획: 전략 보고 고도화

**담당: 비서실 최실장**

### 5-1. 보고서 형식 변경

```
[기존] 숫자만 나열
  "이번 주 지출: 850만원"
  → 사장님이 직접 분석해야 함 ❌

[변경] 원인 + 해결책 포함
  "이번 주 지출 850만원 (전주 대비 +15%)
   원인: 광고비가 200만 → 350만으로 증가
   해결: 광고 채널 B의 ROI가 낮음 → 중단 권고"
  → 사장님은 결정만 하시면 됨 ✅
```

### 5-2. 클로드 활용 보고 프로세스

```
Step 1. 서차장이 주간 장부 엑셀 정리
Step 2. 최실장이 클로드에게 분석 요청
        → "전주 대비 변동 사항 + 원인 + 권고 정리해줘"
Step 3. 클로드가 초안 생성
Step 4. 최실장이 검토 후 사장님께 보고

사장님은 숫자를 보며 고민하실 필요 없음.
"이거 해" / "이건 보류" 결정만 하시면 됩니다.
```

## 9. ERP 콘솔과의 관계

**보고자: 비서실 최실장**

```
Q: ERP 콘솔 개발하고 있는데, Excel + 클로드면 필요 없는 거 아닌가?

A: 아닙니다. 역할이 다릅니다.

┌──────────────────┬──────────────────────┐
│ Claude in Excel  │ ERP 콘솔 (golab)     │
├──────────────────┼──────────────────────┤
│ 기존 엑셀 분석   │ 새로운 업무 시스템    │
│ 과거 데이터 조회 │ 실시간 재고/매입 관리 │
│ 임시 질문/답변   │ 상시 대시보드         │
│ 비정형 분석      │ 정형화된 워크플로우   │
│ 보조 도구        │ 메인 시스템           │
├──────────────────┼──────────────────────┤
│ → 당장 쓸 수 있음│ → 개발 진행 중       │
│ → 기존 장부 활용 │ → 완성 후 메인 전환  │
```
