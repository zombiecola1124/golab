# EP-15 실행문서 ④: 사장님 전용 프롬프트 치트시트

> 용도: 사장님이 엑셀에서 클로드에게 바로 던질 수 있는 질문 모음
> 상태: ★ 사장님 PC 바탕화면에 바로가기 생성할 것 ★
> 비고: 인쇄해서 모니터 옆에 붙여두셔도 됩니다

---

## 사용법

```
1. 엑셀 파일 열기
2. Ctrl+Alt+C (클로드 사이드바 열기)
3. 아래 질문 중 하나를 복사해서 붙여넣기 (또는 직접 입력)
4. 엔터
```

---

## 매일 쓰는 질문 (일상)

```
"오늘 현재 재고 상태 요약해줘"

"재고 부족한 거 있어?"

"이번 달 매입 총액이 얼마야?"

"현금 잔액이 얼마야?"

"가장 최근 납품이 언제야?"
```

---

## 리베이트 / 정산

```
"이번 달 정산 대상자 리스트하고 총액 뽑아"

"미지급 상태인 리베이트가 있어?"

"홍길동 지난 3개월 정산 내역 보여줘"

"이번 분기 리베이트 총액이 얼마지?"
```

---

## 돈 흐름 분석

```
"이번 주 지출이 지난주보다 얼마나 늘었어? 원인이 뭐야?"

"가장 큰 지출 항목 TOP 3 뽑아줘"

"이번 달 입금 예정인 거 있어?"

"다음 달에 돈 부족할 것 같아?"
```

---

## 재고 / 매입

```
"제일 비싼 품목이 뭐야? 재고 얼마나 남았어?"

"3개월간 단가가 많이 오른 품목 있어?"

"A무역에서 올해 총 얼마 사왔어?"

"재고가치 제일 높은 품목 TOP 5"
```

---

## 거래처 분석

```
"A제약한테 올해 얼마 팔았어?"

"거래처별 매출 순위 뽑아줘"

"신규 거래처 언제 들어왔어?"
```

---

## 보고서 / 요약

```
"이번 주 전체 요약 보고서 만들어줘"

"이번 달 수익이 지난달보다 나아졌어 나빠졌어? 원인 분석해줘"

"위험 신호 있으면 알려줘"

"비정상적인 패턴이 있는지 전체 점검해줘"
```

---

## 비교 / 추세

```
"1월 vs 2월 매출 비교해줘"

"지난 6개월간 매입 추세 보여줘"

"어느 달에 돈이 가장 많이 나갔어?"
```

---

## 고급 질문 (사장님 레벨업 후)

```
"마진율이 제일 낮은 품목 찾아서, 단가를 얼마로 올려야 마진 20% 나오는지 계산해줘"

"운임 비중이 원가의 10% 넘는 거래를 찾아줘"

"B업체랑 거래 계속할 가치가 있는지 수치로 판단해줘"

"세금 낼 돈 빼고 실제로 쓸 수 있는 돈이 얼마야?"

"올해 연간 추정 이익이 얼마야? 지금 추세 유지하면?"
```
