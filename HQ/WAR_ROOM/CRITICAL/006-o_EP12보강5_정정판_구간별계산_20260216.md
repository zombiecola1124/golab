## 0. 이전 계산의 오류

```
❌ 잘못된 계산 (보강4):
   이익 1.5억에서 세금 4,077만 + 보험 1,948만 + 리베이트 6,300만 + 와이프 3,600만
   → 세금과 보험을 따로 빼서 계산 → 틀림

✅ 사장님 실제 구조:
   30% 세이브 = 세금 + 건강보험 + 국민연금 + 전부 여기서 냄
   나머지 70%에서 60/40 나눔
   → 세금/보험은 30% 안에 이미 포함
```

## 1. 용어 정리 (사장님 질문 답변)

### 판매장려금이란?

```
= 회사가 거래처한테 "공식적으로" 주는 돈
= 계좌이체로 줌 (현금 아님)
= 회사 비용처리 됨 → 세금 줄어듦
= "법인"이 되어야만 쓸 수 있는 방법

지금 개인사업자 상태에서는 해당 없음.
법인 전환하면 리베이트 일부를 이 명목으로 줄 수 있다는 얘기.
```

### 세금 = 사장님이 30% 세이브하는 그것 맞음

```
30% 세이브 = 종합소득세 + 지방소득세 + 건강보험 + 국민연금 + 장기요양
           = 나라에 내야 하는 것 전부
           = 사장님이 "세금"이라고 부르는 것
```

### 보험 = 건강보험 + 국민연금 + 장기요양

```
개인사업자 지역가입자 기준:
├── 건강보험: 소득 전체 기준으로 부과
├── 국민연금: 소득 기준 (상한 있음)
├── 장기요양보험: 건강보험의 ~13%
└── 전부 사장님 30% 세이브에서 나감
```

## 3. ★ 핵심: 30%가 진짜 충분한가? (이익 구간별)

### 실제 의무 부담 계산

> 종합소득세 + 지방세 + 4대보험(건강+국민+장기요양)
> 소득공제 약 2,500만 적용 (인적공제+보험료공제+기타)

```
┌──────────┬──────────┬──────────┬──────────┬──────────┬──────────┬──────────┐
│ 연 이익   │ 30%      │ 실제     │ 실제     │ 실제     │ 30%로    │ 실제     │
│          │ 세이브    │ 세금     │ 보험     │ 합계     │ 충분?    │ 필요%    │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
│ 8,000만  │ 2,400만  │ 1,150만  │ 1,050만  │ 2,200만  │ ✅ +200  │ 28%     │
│ 1.0억    │ 3,000만  │ 1,720만  │ 1,300만  │ 3,020만  │ ≈ 딱맞음 │ 30%     │
│ 1.2억    │ 3,600만  │ 2,350만  │ 1,550만  │ 3,900만  │ ❌ -300  │ 33%     │
│ 1.5억    │ 4,500만  │ 3,300만  │ 1,950만  │ 5,250만  │ ❌ -750  │ 35%     │
│ 2.0억    │ 6,000만  │ 5,100만  │ 2,400만  │ 7,500만  │ ❌-1,500 │ 38%     │
│ 2.5억    │ 7,500만  │ 7,200만  │ 2,700만  │ 9,900만  │ ❌-2,400 │ 40%     │
│ 3.0억    │ 9,000만  │ 9,500만  │ 2,900만  │ 12,400만 │ ❌-3,400 │ 41%     │
└──────────┴──────────┴──────────┴──────────┴──────────┴──────────┴──────────┘

※ 세금 = 종합소득세 + 지방소득세 (소득공제 ~2,500만, 세액공제 ~50만 반영)
※ 보험 = 건강보험 + 국민연금 + 장기요양 (지역가입자 기준)
※ 성실신고확인 비용 (~200만) 별도

★ 결론:
├── 이익 1억까지: 30% 세이브로 딱 맞음
├── 이익 1.2억 넘으면: 30%로는 부족하기 시작
├── 이익 1.5억이면: 35% 세이브해야 안전
└── 이익 2억 넘으면: 38~40% 필요
```

## 4. ★★ 이익 구간별 전체 분배표

## 6. 법인 전환 시 달라지는 것 (정정판)

```
│                  │                  │                         │

│                  │ (직접 관리)      │                         │
│                  │                  │                         │

│                  │                  │ + 보험 ~560만            │
│                  │                  │ = ~1,080만               │
│                  │                  │                         │

│                  │                  │                         │
```

```
※ 법인 총부담 = 소득세 + 법인세 + 보험(본인+회사) + 유지비(1,800만)

※ 이익 1.5억이면 연 2,370만 절감 = 월 약 200만 추가 생활비
```

## 8. 최종 정리: 사장님이 기억할 것

