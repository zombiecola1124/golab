## 1. 사전 준비 체크리스트

```
□ 사장님 PC에 Microsoft Excel 데스크톱 버전 설치되어 있는지 확인
  → Office 365 또는 Excel 2021 이상 필요
  → 웹 엑셀(Office Online)은 안 됨 ❌
  → 확인법: 엑셀 열어서 [파일] → [계정] → 버전 확인

□ 인터넷 연결 상태 확인
  → 클로드 사이드바가 서버와 통신해야 하므로 인터넷 필수

□ 사장님 이메일 주소 확인 (Claude 계정 가입용)
  → 기존에 claude.ai 계정 있으면 그대로 사용
  → 없으면 새로 만들어야 함
```

---

## 2. Step-by-Step 설치

### Step 1: Claude 계정 가입 + Pro 구독

```
1. 브라우저에서 https://claude.ai 접속
2. [Sign Up] 또는 [Get Started] 클릭
3. 사장님 이메일로 가입
4. 가입 후 왼쪽 하단 [Upgrade to Pro] 클릭
5. Pro 플랜 선택: $20/월 (약 2.7만원)
6. 카드 결제 정보 입력 (사업자 카드 사용 권장)
7. 구독 완료 확인

★ 이미 claude.ai 계정 있으면 → 로그인 후 Pro 업그레이드만 하면 됨
```

### Step 2: Excel에서 Claude 추가 기능 설치

```
[Windows]
1. Excel 열기
2. 상단 [홈] 탭 클릭
3. [추가 기능] 버튼 클릭 (리본 오른쪽 영역)
4. [Office 추가 기능] 선택
5. [스토어] 탭에서 "Claude" 검색
6. "Claude by Anthropic" 찾아서 [추가] 클릭
7. 설치 완료 → 오른쪽에 사이드바 생김

[Mac]
1. Excel 열기
2. 상단 메뉴 [도구] → [추가 기능...] 클릭
3. [스토어] 탭에서 "Claude" 검색
4. "Claude by Anthropic" 찾아서 [추가] 클릭
5. 설치 완료
```

### Step 3: 로그인

```
1. Excel 오른쪽에 Claude 사이드바가 나타남
2. [Sign in] 또는 [로그인] 클릭
3. Step 1에서 만든 Claude 계정으로 로그인
4. 로그인 완료 → 채팅창이 활성화됨
5. 끝!

★ 단축키 (빠른 열기/닫기):
  Windows: Ctrl + Alt + C
  Mac:     Ctrl + Option + C
```

### Step 4: 테스트

```
1. 아무 엑셀 파일 열기 (데이터 있는 파일)
2. 사이드바 채팅창에 입력:
   "이 시트에 무슨 데이터가 있어?"
3. 클로드가 시트 내용을 분석해서 답변하면 → 설치 성공 ✅
4. 추가 테스트:
   "가장 큰 숫자가 어디 있어?"
   "이 데이터 요약해줘"
```

## 5. 문제 해결 (FAQ)

```
Q: 추가 기능에서 Claude가 안 보여요
A: → Office 버전이 오래된 것. Office 365 또는 2021 이상으로 업데이트

Q: 사이드바가 안 열려요
A: → Ctrl+Alt+C 눌러보기. 안 되면 [홈] → [추가 기능] → Claude 다시 선택

Q: 로그인이 안 돼요
A: → claude.ai에서 먼저 로그인 되는지 확인. 비밀번호 재설정 필요할 수 있음

Q: 시트를 못 읽어요
A: → .xlsx 또는 .xlsm 파일인지 확인. .csv나 .xls(구버전)는 변환 필요

Q: 클로드가 엉뚱한 답을 해요
A: → 시트 이름과 열 제목이 명확한지 확인 (장부 표준화 가이드 참조)

Q: 채팅 기록이 사라졌어요
A: → 정상. 엑셀 닫으면 채팅 초기화됨. 중요 분석은 시트에 복사해둘 것
```
