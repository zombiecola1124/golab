## 0. 사장님 구조 (최종 확인)

**보고자: 트럼프 이사**

```
이익 = X (매출에서 사업경비 다 빼고 남은 순이익)
        (펠리세이드 등 사업경비는 이미 빠진 후의 숫자)

① 30% save → 세금 내는 용도 (종소세 + 지방세)
② 나머지 70% 중 → 60% = 리베이트 (친구한테 현금)
                  → 40% = 내 돈 (와이프 주든 뭘하든 내 맘)
③ 건보+연금 → 별도
   → save에서 남는 돈으로 되면 거기서
   → 부족하면 친구한테 "건보 좀 보태줘"
```

## 1. 계산 과정 — 1.5억 기준 상세 풀이

**보고자: 트럼프 이사**

### 1-1. 30% save = 얼마?

```
이익 15,000만원

30% save = 15,000만 × 0.30 = 4,500만원
         = 월 375만원 (4,500 ÷ 12)

→ 이 4,500만원은 통장에 따로 빼놓는 돈
→ 여기서 5월에 세금 냄
```

### 1-2. 나머지 70% = 얼마?

```
나머지 = 15,000만 × 0.70 = 10,500만원
       = 월 875만원 (10,500 ÷ 12)
```

### 1-3. 리베이트 (나머지의 60%) = 얼마?

```
리베이트 = 10,500만 × 0.60 = 6,300만원/연
         = 월 525만원 (6,300 ÷ 12)

→ 친구한테 가는 돈
→ 이익의 42% (= 70% × 60%)
```

### 1-4. 내 돈 (나머지의 40%) = 얼마?

```
내 돈 = 10,500만 × 0.40 = 4,200만원/연
      = 월 350만원 (4,200 ÷ 12)

→ 이익의 28% (= 70% × 40%)
→ 와이프 300만 주면 내 생활비 50만
```

### 1-5. 검산

```
30% save:    4,500만
리베이트:    6,300만
내 돈:       4,200만
─────────────────────
합계:       15,000만 ✅ 맞음
```

## 2. 세금 계산 과정 — 1.5억에서 실제 세금이 얼마인가

**보고자: 트럼프 이사**

### 2-1. 과세표준 구하기

```
이익(사업소득금액):     15,000만원
소득공제:              - 2,000만원 (인적공제+연금공제+보험공제 등)
──────────────────────────────────
과세표준:              13,000만원
```

### 2-2. 종합소득세 계산 (2025년 세율 구간)

```
구간별로 쪼개서 계산합니다:

① 0 ~ 1,400만:     1,400만 × 6%  =    84만
② 1,400 ~ 5,000만: 3,600만 × 15% =   540만
③ 5,000 ~ 8,800만: 3,800만 × 24% =   912만
④ 8,800 ~ 13,000만: 4,200만 × 35% = 1,470만
─────────────────────────────────────────────
종합소득세 합계:                      3,006만원
```

### 2-3. 지방소득세 (종소세의 10%)

```
지방소득세 = 3,006만 × 10% = 약 301만원
```

### 2-4. 세금 총합

```
종합소득세:  3,006만
지방소득세:    301만
──────────────────
세금 합계:   3,307만원/연 (월 276만)
```

### 2-5. 30% save에서 세금 내고 남는 돈

```
30% save:   4,500만
세금:      - 3,307만
──────────────────
남는 돈:    1,193만원 (월 99만)

→ 이 1,193만으로 건보+연금 일부 커버 가능
```

## 3. 건보+연금 계산 과정 — 1.5억 기준

**보고자: 트럼프 이사**

### 3-1. 건강보험료 (지역가입자)

```
개인사업자 = 지역가입자
소득 + 재산 + 차량 점수로 산출

소득 기준 추정:
이익 15,000만 → 소득점수 약 1,100점
점수당 금액: 약 208원 (2025 기준)
월 건강보험료 ≒ 74만원

연 건강보험료 = 74만 × 12 = 886만원/연
```

### 3-2. 장기요양보험료 (건보의 약 12.81%)

```
월 장기요양 = 74만 × 12.81% ≒ 9.5만원
연 장기요양 = 9.5만 × 12 = 113만원/연
```

### 3-3. 국민연금 (지역가입자)

```
국민연금 요율: 소득의 9% (전액 본인 부담)
단, 기준소득월액 상한: 590만원
→ 소득이 아무리 높아도 월 590만 기준으로만 계산

월 국민연금 = 590만 × 9% = 53만원 (상한 고정)
연 국민연금 = 53만 × 12 = 636만원/연

→ 사장님 소득이 1.5억이든 3억이든 연금은 636만 고정
  (상한이 있어서 더 안 올라감)
```

### 3-4. 건보+연금 합계

```
건강보험:     886만원/연 (월 74만)
장기요양:     113만원/연 (월  9만)
국민연금:     636만원/연 (월 53만)
─────────────────────────────────
합계:       1,635만원/연 (월 136만)
```

### 3-5. 건보 재원 — 어디서 내나?

```
save 남는돈:   1,193만원
건보+연금:   - 1,635만원
──────────────────────────
부족분:        442만원/연 (월 37만)

→ 부족한 442만(월 37만)은 친구한테 "건보 좀 보태줘" → 해결
```
