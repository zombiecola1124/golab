## Goal (목표)

- 대성금속 일상 업무(재무·재고·매출·메모)를 **CLI 기반 통합 관리 프로그램**으로 구축
- 첫 MVP: **memo 모듈** 완성 → 대표님이 터미널에서 즉시 메모 CRUD 가능
- 6개월 내 finance + sales + inventory 모듈까지 연동 완료

## Non-Goal (비목표)

- 웹 UI / 모바일 앱은 이번 스코프에서 **제외** (CLI 우선)
- 외부 API 연동(Stripe, 세금 자동화)은 SaaS 매출 확보 후 별도 프로젝트로 진행
- 멀티유저, 권한 관리는 현 단계 불필요 (1인사업자)

## 3대 마일스톤

| # | 마일스톤 | 산출물 | 기한 |
|---|---------|--------|------|
| **M1** | 프로젝트 기반 구축 — `requirements.txt`, `__init__.py`, `tests/` 구조, `main.py` 진입점, 공통 유틸리티 | 실행 가능한 프로젝트 골격 | 1주차 |
| **M2** | memo 모듈 MVP — 메모 CRUD + SQLite 저장 + pytest 테스트 통과 | `memo/` 모듈 동작, 테스트 커버리지 80%+ | 2주차 |
