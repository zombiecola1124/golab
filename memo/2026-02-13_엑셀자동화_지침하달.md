## 배경

사장이 Claude in Excel(Sheets) 기능의 주요 기능을 분석하여 사령부 적용 지침을 하달함.

### 주요 기능 분석 결과
- 자연어 데이터 핸들링: VLOOKUP/피벗 없이 자연어로 시트 병합, 정렬, 요약 가능
- 데이터 정제 및 표준화: 날짜 형식 통일, 오타 정정 등 자동 처리
- 인사이트 자동 도출: 데이터 기반 전략 보고서를 시트 내에서 직접 작성

### 사업 적용 시나리오
- 영업본부(박이사): 고객 상담 일지 분석 → 불만 사항 TOP 3 + 예상 매출 증대 효과
- 재무본부(트럼프이사): 지출 내역 시각화 → 불필요 고정비 누수 실시간 모니터링
